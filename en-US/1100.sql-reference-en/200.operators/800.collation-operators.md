Collation operators 
========================================



The following table lists all collation operators.


| Operator  |    Operand     |  Description   |
|-----------|----------------|----------------|
| `COLLATE` | Unary (suffix) | The collation. |



Examples

```javascript
SELECT last_name
  FROM employees
 ORDER BY last_name COLLATE UTF8MB4_GENERAL_CI;
```
