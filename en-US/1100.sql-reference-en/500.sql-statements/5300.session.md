SESSION 
============================



Description 
--------------------

You can execute the SESSION statement to close a session.

Syntax 
---------------

```javascript
session_stmt:
    KILL [CONNECTION] session_id;

session_id:
    INT_VALUE
```



Parameters 
-------------------



| **Parameter** |                **Description**                |
|---------------|-----------------------------------------------|
| session_id    | The ID of the session that you want to close. |



Examples 
-----------------
