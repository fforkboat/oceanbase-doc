ob_trx_timeout 
===================================

ob_trx_timeout specifies the transaction timeout duration in microseconds. 


|       **Property**        |                                              **Description**                                               |
|---------------------------|------------------------------------------------------------------------------------------------------------|
| Parameter type            | Int                                                                                                        |
| Default value             | 100000000                                                                                                  |
| Value range               | NULL                                                                                                       |
| Effective scope           | * GLOBAL   * SESSION    |
| Involved in serialization | Yes                                                                                                        |
