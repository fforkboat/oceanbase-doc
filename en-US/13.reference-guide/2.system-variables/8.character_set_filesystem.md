character_set_filesystem 
=============================================

character_set_filesystem specifies the character set type of the file system. 


|  **Property**   |                                              **Description**                                               |
|-----------------|------------------------------------------------------------------------------------------------------------|
| Parameter type  | String                                                                                                     |
| Default value   | BINARY                                                                                                     |
| Value range     | BINARY                                                                                                     |
| Effective scope | * GLOBAL   * SESSION    |
| Nullable        | Yes                                                                                                        |
