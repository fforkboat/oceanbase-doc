max_kept_major_version_number 
==================================================

max_kept_major_version_number specifies the number of data versions kept. 


|          **Property**           | **Description** |
|---------------------------------|-----------------|
| Parameter type                  | Integer         |
| Default value                   | 2               |
| Value range                     | \[1, 16\]       |
| Effective upon OBServer restart | No              |
