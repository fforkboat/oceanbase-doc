sys_bkgd_io_timeout 
========================================

sys_bkgd_io_timeout specifies the timeout duration for background I/O requests. 


|          **Property**           | **Description** |
|---------------------------------|-----------------|
| Parameter type                  | Time            |
| Default value                   | 120s            |
| Value range                     | \[5s, 600s\]    |
| Effective upon OBServer restart | No              |
