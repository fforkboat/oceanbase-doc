syslog_level 
=================================

`syslog_level` specifies the level of system logs. 


|          **Property**           |                                                                                                                                         **Description**                                                                                                                                          |
|---------------------------------|--------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------|
| Parameter type                  | String                                                                                                                                                                                                                                                                                           |
| Default value                   | INFO                                                                                                                                                                                                                                                                                             |
| Value range                     | * `DEBUG`   * `TRACE`   * `INFO`   * `WARN`   * `USER_ERR`   * `ERROR`    |
| Effective upon OBServer restart | No                                                                                                                                                                                                                                                                                               |
