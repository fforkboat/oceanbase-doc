clog_disk_usage_limit_percentage 
=====================================================

clog_disk_usage_limit_percentage specifies the maximum disk I/O usage for transaction logs. 


|          **Property**           | **Description** |
|---------------------------------|-----------------|
| Parameter type                  | Integer         |
| Default value                   | 95              |
| Value range                     | \[80, 100\]     |
| Effective upon OBServer restart | No              |
