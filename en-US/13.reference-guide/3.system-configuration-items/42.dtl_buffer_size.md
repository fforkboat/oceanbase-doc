dtl_buffer_size 
====================================

dtl_buffer_size specifies the cache size allocated to the SQL data transfer module. 


|          **Property**           | **Description** |
|---------------------------------|-----------------|
| Parameter type                  | Capacity unit   |
| Default value                   | 64K             |
| Value range                     | \[4K, 2 M\]     |
| Effective upon OBServer restart | No              |
