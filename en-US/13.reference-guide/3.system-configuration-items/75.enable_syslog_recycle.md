enable_syslog_recycle 
==========================================

enable_syslog_recycle specifies whether to enable system log recycling. 


|          **Property**           |                                            **Description**                                             |
|---------------------------------|--------------------------------------------------------------------------------------------------------|
| Parameter type                  | Boolean                                                                                                |
| Default value                   | False                                                                                                  |
| Value range                     | * True   * False    |
| Effective upon OBServer restart | No                                                                                                     |
