enable_syslog_wf 
=====================================

enable_syslog_wf specifies whether to print system logs above the WARNING level to a separate log file. 


|          **Property**           |                                            **Description**                                             |
|---------------------------------|--------------------------------------------------------------------------------------------------------|
| Parameter type                  | Boolean                                                                                                |
| Default value                   | True                                                                                                   |
| Value range                     | * True   * False    |
| Effective upon OBServer restart | No                                                                                                     |
