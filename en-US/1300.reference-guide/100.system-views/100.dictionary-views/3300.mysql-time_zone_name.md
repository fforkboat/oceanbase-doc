mysql.time_zone_name 
=========================================





Overview 
--------------------------

mysql.time_zone_name provides time zone information (SYSTEM_VIEW).

|  Field name  |    Type     | Nullable |        Description         |
|--------------|-------------|----------|----------------------------|

| Time_zone_id | bigint(20)  | NO       | The ID of the time zone.   |
