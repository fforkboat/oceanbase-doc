validate_password_length 
=============================================

validate_password_length specifies the minimum length of a user account password. 


|       **Property**        | **Description**  |
|---------------------------|------------------|
| Parameter type            | UInt             |
| Default value             | 0                |
| Value range               | \[0,2147483647\] |
| Effective scope           | GLOBAL           |
| Involved in serialization | Yes              |
