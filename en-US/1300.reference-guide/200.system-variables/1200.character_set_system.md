character_set_system 
=========================================
`character_set_system` specifies the character set used by the server. 
|  **Property**   |                                              **Description**                                               |
|-----------------|------------------------------------------------------------------------------------------------------------|
| Parameter type  | String                                                                                                     |
| Default value   | utf8mb4                                                                                                    |
| Value range     | N/A                                                                                                   |
| Effective scope | <ul><li> Global   </li><li> Session   </li></ul>   |
| Modifiable      | No                                                                                                         |
