version_comment 
====================================

version_comment specifies the version information of OBServer. 


|  **Property**   | **Description** |
|-----------------|-----------------|
| Parameter type  | Varchar         |
| Default value   | OceanBase 1.0.0 |
| Value range     | N/A             |
| Effective scope | GLOBAL          |
| Modifiable      | No              |
