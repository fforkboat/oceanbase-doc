auto_increment_offset 
==========================================

auto_increment_offset specifies the starting value for AUTO_INCREMENT. 


|  **Property**   |                                              **Description**                                               |
|-----------------|------------------------------------------------------------------------------------------------------------|
| Parameter type  | UInt                                                                                                       |
| Default value   | 1                                                                                                          |
| Value range     | \[1, 65535\]                                                                                               |
| Effective scope | * GLOBAL   * SESSION    |
