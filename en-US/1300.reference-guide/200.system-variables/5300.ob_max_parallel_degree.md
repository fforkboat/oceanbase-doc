ob_max_parallel_degree 
===========================================

ob_max_parallel_degree specifies the maximum number of concurrent operations per request. 


|  **Property**   |                                              **Description**                                               |
|-----------------|------------------------------------------------------------------------------------------------------------|
| Parameter type  | Int                                                                                                        |
| Default value   | 16                                                                                                         |
| Value range     | NULL                                                                                                       |
| Effective scope | * GLOBAL   * SESSION    |
