ob_query_timeout 
=====================================

ob_query_timeout specifies the query timeout duration in microseconds. 


|       **Property**        |                                              **Description**                                               |
|---------------------------|------------------------------------------------------------------------------------------------------------|
| Parameter type            | Int                                                                                                        |
| Default value             | 10000000                                                                                                   |
| Value range               | NULL                                                                                                       |
| Effective scope           | * GLOBAL   * SESSION    |
| Involved in serialization | Yes                                                                                                        |
