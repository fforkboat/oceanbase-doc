ob_plan_cache_evict_high_percentage 
========================================================

ob_plan_cache_evict_high_percentage specifies the memory usage to trigger plan cache eviction. 


|       **Property**        | **Description** |
|---------------------------|-----------------|
| Parameter type            | Int             |
| Default value             | 90              |
| Value range               | \[0, 100\]      |
| Effective scope           | GLOBAL          |
| Involved in serialization | Yes             |
