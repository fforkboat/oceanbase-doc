ob_sql_work_area_percentage 
================================================

ob_sql_work_area_percentage specifies the maximum percentage of tenant memory for SQL execution. 


|  **Property**   | **Description** |
|-----------------|-----------------|
| Parameter type  | Int             |
| Default value   | 5               |
| Value range     | \[0, 100\]      |
| Effective scope | GLOBAL          |
