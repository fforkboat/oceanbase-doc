sql_throttle_network 
=========================================

sql_throttle_network specifies the request queuing time. If the request queuing time is greater than or equal to the specified value, the query is not executed.

|-----------------|-------------------------------------------------------------------|
| Parameter type  | numeric                                                           |
| Default value   | -1                                                                |
