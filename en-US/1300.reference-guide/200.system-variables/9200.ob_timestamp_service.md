ob_timestamp_service 
=========================================

ob_timestamp_service specifies the timestamp service. 


|       **Property**        |                                                                 **Description**                                                                 |
|---------------------------|-------------------------------------------------------------------------------------------------------------------------------------------------|
| Parameter type            | Enum                                                                                                                                            |
| Default value             | GTS                                                                                                                                             |
| Value range               | * LTS   * GTS   * HA_GTS    |
| Effective scope           | GLOBAL                                                                                                                                          |
| Involved in serialization | Yes                                                                                                                                             |
