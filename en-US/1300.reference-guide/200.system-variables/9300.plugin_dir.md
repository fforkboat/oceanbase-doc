plugin_dir 
===============================

plugin_dir specifies the path for storing plug-in DLL files. 


|  **Property**   |                                              **Description**                                               |
|-----------------|------------------------------------------------------------------------------------------------------------|
| Parameter type  | Varchar                                                                                                    |
| Default value   | ./plugin_dir/                                                                                              |
| Value range     | NULL                                                                                                       |
| Effective scope | * GLOBAL   * SESSION    |
