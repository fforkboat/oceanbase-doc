parallel_max_servers 
=========================================

parallel_max_servers specifies the size of the Parallel eXecution (PX) thread pool on each server. 


|       **Property**        | **Description** |
|---------------------------|-----------------|
| Parameter type            | Int             |
| Default value             | 0               |
| Value range               | \[0, 1800\]     |
| Effective scope           | GLOBAL          |
| Involved in serialization | Yes             |
