internal_sql_execute_timeout 
=================================================

internal_sql_execute_timeout specifies the interval of DML requests in the system. 


|          **Property**           |  **Description**  |
|---------------------------------|-------------------|
| Parameter type                  | Time              |
| Default value                   | 30s               |
| Value range                     | \[1000us, 10min\] |
| Effective upon OBServer restart | No                |
