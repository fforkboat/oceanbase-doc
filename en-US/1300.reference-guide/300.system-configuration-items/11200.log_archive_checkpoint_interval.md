log_archive_checkpoint_interval 
====================================================

`log_archive_checkpoint_interval` specifies the interval between log archive checkpoints of cold data. 


|          **Property**           | **Description** |
|---------------------------------|-----------------|
| Parameter type                  | Time            |
| Default value                   | 120s            |
| Value range                     | \[5s, 1h\]      |
| Effective upon OBServer restart | No              |
