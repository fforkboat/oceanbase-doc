major_freeze_duty_time 
===========================================

major_freeze_duty_time specifies the time for triggering a daily freeze and compaction. 


|          **Property**           | **Description**  |
|---------------------------------|------------------|
| Parameter type                  | Time             |
| Default value                   | 02:00            |
| Value range                     | \[00:00, 23:59\] |
| Effective upon OBServer restart | No               |
