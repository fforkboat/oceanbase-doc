merger_warm_up_duration_time 
=================================================

merger_warm_up_duration_time specifies the preload time of new baseline data in a major compaction. 


|          **Property**           | **Description** |
|---------------------------------|-----------------|
| Parameter type                  | Time            |
| Default value                   | 0s              |
| Value range                     | \[0s, 60m\]     |
| Effective upon OBServer restart | No              |
