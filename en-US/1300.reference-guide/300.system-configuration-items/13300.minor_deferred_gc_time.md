minor_deferred_gc_time 
===========================================

minor_deferred_gc_time specifies the interval of deferred garbage collection for SSTables after a major compaction. 


|          **Property**           | **Description** |
|---------------------------------|-----------------|
| Parameter type                  | Time            |
| Default value                   | 0s              |
| Value range                     | \[0s, 24h\]     |
| Effective upon OBServer restart | No              |
