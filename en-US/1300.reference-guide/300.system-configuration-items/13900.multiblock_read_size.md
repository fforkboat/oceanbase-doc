multiblock_read_size 
=========================================

multiblock_read_size specifies the I/O aggregate throughput in data access. 


|          **Property**           | **Description** |
|---------------------------------|-----------------|
| Parameter type                  | Capacity unit   |
| Default value                   | 128K            |
| Value range                     | \[0K, 2M\]      |
| Effective upon OBServer restart | No              |
