px_workers_per_cpu_quota 
=============================================

px_workers_per_cpu_quota specifies the ratio of parallel worker threads. 


|          **Property**           | **Description** |
|---------------------------------|-----------------|
| Parameter type                  | Integer         |
| Default value                   | 10              |
| Value range                     | \[0, 20\]       |
| Effective upon OBServer restart | No              |
