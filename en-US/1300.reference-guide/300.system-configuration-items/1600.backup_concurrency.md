backup_concurrency 
=======================================

backup_concurrency specifies the number of concurrent tasks of writing data to the file system during backup. 


|          **Property**           | **Description** |
|---------------------------------|-----------------|
| Parameter type                  | Integer         |
| Default value                   | 0               |
| Value range                     | \[0, 100\]      |
| Effective upon OBServer restart | No              |
