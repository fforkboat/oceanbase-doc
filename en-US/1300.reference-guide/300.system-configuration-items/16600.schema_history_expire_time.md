schema_history_expire_time 
===============================================

schema_history_expire_time specifies the lifetime of history metadata. 


|          **Property**           |      **Description**      |
|---------------------------------|---------------------------|
| Parameter type                  | Time                      |
| Default value                   | 7d, indicating seven days |
| Value range                     | \[1h, 30d\]               |
| Effective upon OBServer restart | No                        |
