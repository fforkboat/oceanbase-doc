stack_size 
===============================

stack_size specifies the size of the function call stack for programs. 


|          **Property**           | **Description** |
|---------------------------------|-----------------|
| Parameter type                  | Capacity unit   |
| Default value                   | 1M              |
| Value range                     | \[512K, 20M\]   |
| Effective upon OBServer restart | No              |
