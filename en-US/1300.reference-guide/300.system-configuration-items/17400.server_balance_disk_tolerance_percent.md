server_balance_disk_tolerance_percent 
==========================================================

server_balance_disk_tolerance_percent specifies the tolerance of imbalance in the disk load balancing policies of different servers. 


|          **Property**           | **Description** |
|---------------------------------|-----------------|
| Parameter type                  | Integer         |
| Default value                   | 1               |
| Value range                     | \[1, 100\]      |
| Effective upon OBServer restart | No              |
