sys_bkgd_io_high_percentage 
================================================

sys_bkgd_io_high_percentage sets the highest percentage of traffic for background I/O tasks. 


|          **Property**           | **Description** |
|---------------------------------|-----------------|
| Parameter type                  | Integer         |
| Default value                   | 90              |
| Value range                     | \[1, 100\]      |
| Effective upon OBServer restart | No              |
