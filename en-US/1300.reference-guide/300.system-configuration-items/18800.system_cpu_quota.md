system_cpu_quota 
=====================================

system_cpu_quota specifies the CPU quota available to the system tenant. 


|          **Property**           |            **Description**             |
|---------------------------------|----------------------------------------|
| Parameter type                  | Double-precision floating-point number |
| Default value                   | 10                                     |
| Value range                     | \[0, 16\]                              |
| Effective upon OBServer restart | No                                     |
