trx_2pc_retry_interval 
===========================================

trx_2pc_retry_interval specifies the time interval at which a failed two-phase commit is retried. 


|          **Property**           | **Description** |
|---------------------------------|-----------------|
| Parameter type                  | Time            |
| Default value                   | 100ms           |
| Value range                     | \[1ms, 5000ms\] |
| Effective upon OBServer restart | No              |
