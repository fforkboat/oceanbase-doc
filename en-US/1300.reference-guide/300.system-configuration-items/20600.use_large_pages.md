use_large_pages 
====================================

use_large_pages manages the use of large memory pages by the database. 


|          **Property**           |                                                                 **Description**                                                                  |
|---------------------------------|--------------------------------------------------------------------------------------------------------------------------------------------------|
| Parameter type                  | String                                                                                                                                           |
| Default value                   | False                                                                                                                                            |
| Value range                     | * False   * True   * Only    |
| Effective upon OBServer restart | Yes                                                                                                                                              |
