backup_log_archive_checkpoint_interval 
===========================================================

backup_log_archive_checkpoint_interval specifies the time interval for backing up data backup archivelogs. 


|            Property             | Description |
|---------------------------------|-------------|
| Parameter type                  | Time        |
| Default value                   | 0           |
| Value range                     | \[0s, 24h\] |
| Effective upon OBServer restart | No          |
