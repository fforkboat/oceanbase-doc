open_cursors 
=================================

open_cursors specifies the maximum number of cursors that can be opened at one time in a single session. 


|            Attribute            | Description |
|---------------------------------|-------------|
| Type                            | Integer     |
| Default value                   | 50          |
| Value range                     | \[0,65535\] |
| Effective upon OBServer restart | No          |
