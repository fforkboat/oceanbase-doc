cpu_reserved 
=================================

cpu_reserved specifies the number of CPUs reserved in the system. The remaining CPUs are exclusively occupied by OceanBase. 


|          **Property**           | **Description** |
|---------------------------------|-----------------|
| Parameter type                  | Integer         |
| Default value                   | 2               |
| Value range                     | \[0, 15\]       |
| Effective upon OBServer restart | No              |
