enable_one_phase_commit 
============================================

`enable_one_phase_commit` specifies whether to enable one-phase commit. 


|          **Property**           |                                            **Description**                                             |
|---------------------------------|--------------------------------------------------------------------------------------------------------|
| Parameter type                  | Boolean                                                                                                |
| Default value                   | False                                                                                              |
| Value range                     | * True   * False    |
| Effective upon OBServer restart | No                                                                                                     |
