enable_election_group 
==========================================

enable_election_group specifies whether to enable the election group policy. 


|            Property             |                                              Description                                               |
|---------------------------------|--------------------------------------------------------------------------------------------------------|
| Parameter type                  | Boolean                                                                                                |
| Default value                   | True                                                                                                   |
| Value range                     | * True   * False    |
| Effective upon OBServer restart | No                                                                                                     |
