enable_merge_by_turn 
=========================================

`enable_merge_by_turn` specifies whether to enable the rotating compaction policy. 


|          **Property**           |                                            **Description**                                             |
|---------------------------------|--------------------------------------------------------------------------------------------------------|
| Parameter type                  | Boolean                                                                                                |
| Default value                   | False                                                                             |
| Value range                     | * True   * False    |
| Effective upon OBServer restart | No                                                                                                     |
