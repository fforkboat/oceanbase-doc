enable_major_freeze 
========================================

enable_major_freeze specifies whether to enable automatic global freezing. 


|          **Property**           |                                            **Description**                                             |
|---------------------------------|--------------------------------------------------------------------------------------------------------|
| Parameter type                  | Boolean                                                                                                |
| Default value                   | True                                                                                                   |
| Value range                     | * True   * False    |
| Effective upon OBServer restart | No                                                                                                     |
