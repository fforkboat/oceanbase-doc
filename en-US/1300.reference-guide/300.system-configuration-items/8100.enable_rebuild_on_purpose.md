enable_rebuild_on_purpose 
==============================================

enable_rebuild_on_purpose specifies whether to enable automatic replica reconstruction when the logs of the backup replica lag far behind. 


|          **Property**           |                                            **Description**                                             |
|---------------------------------|--------------------------------------------------------------------------------------------------------|
| Parameter type                  | Boolean                                                                                                |
| Default value                   | False                                                                                                  |
| Value range                     | * True   * False    |
| Effective upon OBServer restart | No                                                                                                     |
