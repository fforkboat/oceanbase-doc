enable_monotonic_weak_read 
===============================================

enable_monotonic_weak_read specifies whether to enable monotonic reads. 


|          **Property**           |                                            **Description**                                             |
|---------------------------------|--------------------------------------------------------------------------------------------------------|
| Parameter type                  | Boolean                                                                                                |
| Default value                   | False                                                                                                  |
| Value range                     | * True   * False    |
| Effective upon OBServer restart | No                                                                                                     |
