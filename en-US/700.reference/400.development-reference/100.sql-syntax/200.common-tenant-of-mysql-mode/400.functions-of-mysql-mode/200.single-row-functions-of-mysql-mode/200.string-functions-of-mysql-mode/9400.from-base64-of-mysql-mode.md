# FROM_BASE64

## Syntax

```sql
FROM_BASE64(str)
```

## Purpose

`FROM_BASE64()` decodes a Base64-encoded string and returns it as a binary string. It is usually used to decode a string encoded by the [TO_BASE64()](93.to-base64-mysql-mode.md) function.

## Examples

```sql
obclient [test]> SELECT FROM_BASE64('T0I=');
+---------------------+
| FROM_BASE64('T0I=') |
+---------------------+
| OB                  |
+---------------------+
1 row in set
```
