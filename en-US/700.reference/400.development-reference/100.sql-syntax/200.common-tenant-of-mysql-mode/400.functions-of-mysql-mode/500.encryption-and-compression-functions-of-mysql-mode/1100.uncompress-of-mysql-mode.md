# UNCOMPRESS

## Syntax

```sql
UNCOMPRESS(str)
```

## Purpose

`UNCOMPRESS()` decompresses a string compressed by using the [COMPRESS()](3.compress-of-mysql-mode.md) function.

## Examples

```sql
obclient [test]> SELECT UNCOMPRESS(COMPRESS('abc'));
+-----------------------------+
| UNCOMPRESS(COMPRESS('abc')) |
+-----------------------------+
| abc                         |
+-----------------------------+
1 row in set
```
