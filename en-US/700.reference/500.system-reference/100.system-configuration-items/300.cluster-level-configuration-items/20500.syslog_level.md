# syslog_level

`syslog_level` specifies the level of system logs.

| **Property** | **Description** |
| --- | --- |
| Type | String |
| Default value | WDIAG |
| Valid values | <ul><li>  DEBUG </li><li> TRACE</li><li>WDIAG</li><li>EDIAG</li><li>INFO </li><li> WARN</li><li> ERROR</li></ul> |
| Effective upon an OBServer restart | No |
