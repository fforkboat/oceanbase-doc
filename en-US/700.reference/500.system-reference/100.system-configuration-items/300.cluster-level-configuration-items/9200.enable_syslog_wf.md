enable_syslog_wf
=====================================

`enable_syslog_wf` specifies whether to print system logs above the WARN level to a separate log file.


| **Parameter** | **Description** |
|------------------|--------------------------------------------------------------------------------------------------------|
| Type | Boolean |
| Default value | True |
| Value range | <li> True   <li> False |
| Effective upon restart of OBServer node | No |
