# server_uuid

`server_uuid` shows the UUID of an OBServer node.

A UUID is generated automatically when an OBServer node starts. Example: `182d0ebb-1209-11ed-b01e-a237b1d47168`.

| **Property** | **Description** |
|---------|------------------|
| Type | Varchar |
| Default value | N/A |
| Valid values | N/A |
| Applicable scope | Global |
| Modifiable | No |
