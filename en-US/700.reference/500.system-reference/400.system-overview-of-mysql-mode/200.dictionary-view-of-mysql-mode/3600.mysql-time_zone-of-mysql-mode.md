mysql.time_zone
====================================


Feature
-----------

This view provides information about time zones.

Fields
-------------

| **Field** | **Type** | **Nullable** | **Description** |
|------------------|------------|-----------------|--------|
| Time_zone_id | bigint(20) | NO | The ID of the time zone. |
| Use_leap_seconds | varchar(8) | NO | Indicates whether leap seconds are used. |
