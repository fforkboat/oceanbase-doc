session_track_schema 
=========================================

`session_track_schema` 用于控制返回的 OK 包中是否包含 schema 变更的信息。


| **属性** |                                                   **描述**                                                   |
|--------|------------------------------------------------------------------------------------------------------------|
| 参数类型   | bool                                                                                                       |
| 默认值    | 1                                                                                                          |
| 取值范围   | <ul><li>0</li><li>1</li></ul>              |
| 生效范围   | <ul><li>Global</li><li>Session</li></ul>    |
