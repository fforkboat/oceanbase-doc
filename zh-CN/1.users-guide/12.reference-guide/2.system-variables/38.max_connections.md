max_connections 
====================================
#docslug#/oceanbase-database/oceanbase-database/V4.0.0/max_connections
`max_connections` 用于设置允许使用的 FD 最大值。


|  **属性**  |      **描述**      |
|----------|------------------|
| 参数类型     | uint             |
| 默认值      | 2147483647       |
| 取值范围     | \[1,2147483647\] |
| 生效范围     | Global           |
| 是否参与序列化  | 否                |
| 是否影响计划生成 | 否                |
| 是否可以为空   | 否                |
| 是否可修改    | 是                |
