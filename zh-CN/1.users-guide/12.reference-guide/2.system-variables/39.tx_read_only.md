tx_read_only 
=================================

`tx_read_only` 用于设置是否为只读事务。


| **属性**  |                                                   **描述**                                                    |
|---------|-------------------------------------------------------------------------------------------------------------|
| 参数类型    | int                                                                                                         |
| 默认值     | 0                                                                                                           |
| 取值范围    | <ul><li> 1：只读事务  </li><li> 0：不是只读事务</li></ul>    |
| 生效范围    | <ul><li>Global</li><li>Session</li></ul>     |
| 是否参与序列化 | 是                                                                                                           |


> **说明**<br>
> 设置后，仅影响该用户，不影响其他用户。
