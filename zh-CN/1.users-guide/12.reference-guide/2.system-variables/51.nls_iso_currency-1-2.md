nls_iso_currency 
=====================================
#docslug#/oceanbase-database/oceanbase-database/V4.0.0/nls_iso_currency-1-2
`nls_iso_currency` 用于设置数字格式元素 `C` 的本地货币符号。

> **说明**
> 
> 该变量当前暂不生效。

|   属性    |                                                     描述                                                     |
|---------|------------------------------------------------------------------------------------------------------------|
| 参数类型    | varchar                                                                                                    |
| 默认值     | AMERICA                                                                                                    |
| 取值范围    | N/A                                                                                                        |
| 生效范围    | <li> Global   <li> Session    |
| 是否参与序列化 | 是，即远程执行、分布式执行场景下，变量会参与序列化。                                                                                 |
