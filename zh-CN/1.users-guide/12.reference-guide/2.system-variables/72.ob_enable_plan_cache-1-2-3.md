ob_enable_plan_cache 
=========================================
#docslug#/oceanbase-database/oceanbase-database/V4.0.0/ob_enable_plan_cache-1-2-3
ob_enable_plan_cache 用于设置是否打开 Plan Cache。


| **属性**  |                                                   **描述**                                                   |
|---------|------------------------------------------------------------------------------------------------------------|
| 参数类型    | bool                                                                                                       |
| 默认值     | 1                                                                                                          |
| 取值范围    | <li> 0：关闭   <li> 1：打开         |
| 生效范围    | <li> GLOBAL   <li> SESSION    |
| 是否参与序列化 | 是                                                                                                          |
