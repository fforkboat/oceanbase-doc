ob_route_policy 
====================================

`ob_route_policy` 用于设置 OBProxy 或 Java 客户端与 OBServer 内部重试的路由策略。


|  **属性**  |                                                                                                                             **描述**                                                                                                                             |
|----------|----------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------|
| 参数类型     | enum                                                                                                                                                                                                                                                           |
| 默认值      | 1                                                                                                                                                                                                                                                              |
| 取值范围     | <ul><li>1：READONLY_ZONE_FIRST </li><li> 2：ONLY_READONLY_ZONE </li><li> 3：UNMERGE_ZONE_FIRST</li><li> 4：UNMERGE_FOLLOWER_FIRST</li></ul>    |
| 生效范围     | <ul><li>Global</li><li>Session</li></ul>                                                                                                                                                        |
| 是否参与序列化  | 是                                                                                                                                                                                                                                                              |
| 是否影响计划生成 | 是                                                                                                                                                                                                                                                              |
