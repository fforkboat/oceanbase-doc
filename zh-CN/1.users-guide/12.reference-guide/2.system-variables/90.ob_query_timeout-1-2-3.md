ob_query_timeout 
=====================================
#docslug#/oceanbase-database/oceanbase-database/V4.0.0/ob_query_timeout-1-2-3
`ob_query_timeout` 用于设置对 SQL 语句进行 DML 操作的超时时间，单位是微秒。


| **属性**  |                                                   **描述**                                                   |
|---------|------------------------------------------------------------------------------------------------------------|
| 参数类型    | int                                                                                                        |
| 默认值     | 10000000                                                                                                   |
| 取值范围    | \[0, 3216672000000000\]                                                                                    |
| 生效范围    | <li> Global   <li> Session    |
| 是否参与序列化 | 是                                                                                                          |
