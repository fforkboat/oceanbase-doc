ignore_replay_checksum_error 
=================================================
#docslug#/oceanbase-database/oceanbase-database/V4.0.0/ignore_replay_checksum_error-1
`ignore_replay_checksum_error` 用于设置是否忽略回放事务日志时发生的校验和错误。


|      **属性**      |                                                 **描述**                                                 |
|------------------|--------------------------------------------------------------------------------------------------------|
| 参数类型             | 布尔类型                                                                                                   |
| 默认值              | False                                                                                                   |
| 取值范围             | <li>True   <li>False    |
| 是否重启 OBServer 生效 | 否                                                                                                      |
