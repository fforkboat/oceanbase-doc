max_px_worker_count 
========================================
#docslug#/oceanbase-database/oceanbase-database/V4.0.0/max_px_worker_count-1-2-3
max_px_worker_count 用于设置 SQL 并行查询引擎使用的最大线程数。


|   **属** **性**    |    **描述**    |
|------------------|--------------|
| 参数类型             | 整型           |
| 默认值              | 64           |
| 取值范围             | \[0, 65535\] |
| 是否重启 OBServer 生效 | 否            |
