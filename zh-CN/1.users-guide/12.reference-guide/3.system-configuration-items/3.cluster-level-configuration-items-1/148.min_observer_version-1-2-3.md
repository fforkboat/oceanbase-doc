min_observer_version 
=========================================
#docslug#/oceanbase-database/oceanbase-database/V4.0.0/min_observer_version-1-2-3
`min_observer_version` 用于查看本集群最低的 OBServer 应用程序版本号。


|      **属性**      | **描述**  |
|------------------|---------|
| 参数类型             | 字符串     |
| 默认值              | 4.0.0.0 |
| 取值范围             | NULL    |
| 是否重启 OBServer 生效 | 否       |
