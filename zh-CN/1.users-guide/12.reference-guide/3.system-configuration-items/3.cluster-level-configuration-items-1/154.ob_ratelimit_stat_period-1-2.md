ob_ratelimit_stat_period 
=============================================
#docslug#/oceanbase-database/oceanbase-database/V4.0.0/ob_ratelimit_stat_period-1-2
`ob_ratelimit_stat_period` 用于设置 OBServer 计算和更新最大带宽的时间间隔。


|        属性        |     描述      |
|------------------|-------------|
| 参数类型             | 时间类型        |
| 默认值              | 1s          |
| 取值范围             | \[100ms,+∞) |
| 是否重启 OBServer 生效 | 否           |
