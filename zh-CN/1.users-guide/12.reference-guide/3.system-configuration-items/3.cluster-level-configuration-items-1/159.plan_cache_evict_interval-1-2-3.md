plan_cache_evict_interval 
==============================================
#docslug#/oceanbase-database/oceanbase-database/V4.0.0/plan_cache_evict_interval-1-2-3
plan_cache_evict_interval 用于设置执行计划缓存的淘汰时间间隔。


|      **属性**      |  **描述**   |
|------------------|-----------|
| 参数类型             | 时间类型      |
| 默认值              | 30s       |
| 取值范围             | \[0s, +∞) |
| 是否重启 OBServer 生效 | 否         |
