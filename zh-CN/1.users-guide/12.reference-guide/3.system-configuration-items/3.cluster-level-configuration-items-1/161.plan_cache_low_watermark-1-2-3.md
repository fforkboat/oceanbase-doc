plan_cache_low_watermark 
=============================================
#docslug#/oceanbase-database/oceanbase-database/V4.0.0/plan_cache_low_watermark-1-2-3
plan_cache_low_watermark 用于设置执行计划缓存占用内存的阈值，如果低于该阈值时将停止淘汰。

> **说明**
> 
> 在当前版本中，该配置项已不再使用。


|      **属性**      | **描述** |
|------------------|--------|
| 参数类型             | 容量单位   |
| 默认值              | 1500M  |
| 取值范围             | NULL   |
| 是否重启 OBServer 生效 | 否      |
