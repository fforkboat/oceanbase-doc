px_task_size 
=================================
#docslug#/oceanbase-database/oceanbase-database/V4.0.0/px_task_size-1-2-3
px_task_size 用于设置 SQL 并行查询引擎每个任务处理的数据量大小。


|      **属性**      |  **描述**   |
|------------------|-----------|
| 参数类型             | 容量单位      |
| 默认值              | 2M        |
| 取值范围             | \[2M, +∞) |
| 是否重启 OBServer 生效 | 否         |
