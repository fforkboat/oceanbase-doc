syslog_level 
=================================
#docslug#/oceanbase-database/oceanbase-database/V4.0.0/syslog_level-1-2-3
`syslog_level` 用于设置系统日志级别。


|      **属性**      |                                                                                                                                        **描述**                                                                                                                                        |
|------------------|--------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------|
| 参数类型             | 字符串                                                                                                                                                                                                                                                                                  |
| 默认值              | INFO                                                                                                                                                                                                                                                                                 |
| 取值范围             | <li> DEBUG   <li> TRACE   <li> INFO   <li> WARN   <li> USER_ERR   <li> ERROR    |
| 是否重启 OBServer 生效 | 否                                                                                                                                                                                                                                                                                    |
