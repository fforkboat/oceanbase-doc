tcp_keepidle 
=================================
#docslug#/oceanbase-database/oceanbase-database/V4.0.0/tcp_keepidle-1-2-3
`tcp_keepidle` 用于设置客户端连接上服务器后，如果没有数据发送，多久后会发送 Keepalive 探测分组，单位为秒。


|        属性        |    描述    |
|------------------|----------|
| 参数类型             | 时间类型     |
| 默认值              | 7200s    |
| 取值范围             | \[1s,+∞) |
| 是否重启 OBServer 生效 | 否        |
