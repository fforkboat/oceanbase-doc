trx_2pc_retry_interval 
===========================================
#docslug#/oceanbase-database/oceanbase-database/V4.0.0/trx_2pc_retry_interval-1-2-3
trx_2pc_retry_interval 用于设置两阶段提交失败时候自动重试的时间间隔。


|      **属性**      |     **描述**      |
|------------------|-----------------|
| 参数类型             | 时间类型            |
| 默认值              | 100ms           |
| 取值范围             | \[1ms, 5000ms\] |
| 是否重启 OBServer 生效 | 否               |
