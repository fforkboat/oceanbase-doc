weak_read_version_refresh_interval 
=======================================================
#docslug#/oceanbase-database/oceanbase-database/V4.0.0/weak_read_version_refresh_interval-1-2-3
`weak_read_version_refresh_interval` 用于设置弱一致性读版本号的刷新周期，影响弱一致性读数据的延时。


|      **属性**      |   **描述**    |
|------------------|-------------|
| 参数类型             | 时间类型        |
| 默认值              | 100ms          |
| 取值范围             | \[50ms, +∞) |
| 是否重启 OBServer 生效 | 否           |
