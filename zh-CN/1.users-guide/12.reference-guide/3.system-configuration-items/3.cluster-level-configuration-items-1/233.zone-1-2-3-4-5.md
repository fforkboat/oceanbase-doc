zone 
=========================
#docslug#/oceanbase-database/oceanbase-database/V4.0.0/zone-1-2-3-4-5
zone 用于设置节点所在的 Zone 的名字。该配置项一般不需要配置。


|      **属性**      | **描述** |
|------------------|--------|
| 参数类型             | 字符串类型  |
| 默认值              | 空值     |
| 取值范围             | NULL   |
| 是否重启 OBServer 生效 | 否      |
