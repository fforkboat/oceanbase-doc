cluster_id 
===============================
#docslug#/oceanbase-database/oceanbase-database/V4.0.0/cluster_id-1-2
`cluster_id` 用于设置本 OceanBase 集群的 ID。

> **说明**
> 
> 该配置项仅支持在部署集群时设置，集群生效后，无法修改。


|      **属性**      |      **描述**       |
|------------------|-------------------|
| 参数类型             | 整型                |
| 默认值              | 0                 |
| 取值范围             | \[1, 4294901759\] |
| 是否重启 OBServer 生效 | 否                 |
