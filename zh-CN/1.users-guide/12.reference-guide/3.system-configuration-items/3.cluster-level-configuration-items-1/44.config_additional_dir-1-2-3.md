config_additional_dir 
==========================================
#docslug#/oceanbase-database/oceanbase-database/V4.0.0/config_additional_dir-1-2-3
config_additional_dir 用于设置本地存储配置文件的多个目录，为了冗余存储多份配置文件。


|      **属性**      |                                                **描述**                                                 |
|------------------|-------------------------------------------------------------------------------------------------------|
| 参数类型             | 字符串列表                                                                                                 |
| 默认值              | <li> etc2   <li> etc3    |
| 取值范围             | NULL                                                                                                  |
| 是否重启 OBServer 生效 | 否                                                                                                     |
