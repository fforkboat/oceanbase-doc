enable_ddl 
===============================
#docslug#/oceanbase-database/oceanbase-database/V4.0.0/enable_ddl-1-2
enable_ddl 用于设置是否允许执行 DDL。


|      **属性**      |                                                   描述                                                   |
|------------------|--------------------------------------------------------------------------------------------------------|
| 参数类型             | 布尔类型                                                                                                   |
| 默认值              | True                                                                                                   |
| 取值范围             | <li> True   <li> False    |
| 是否重启 OBServer 生效 | 否                                                                                                      |
