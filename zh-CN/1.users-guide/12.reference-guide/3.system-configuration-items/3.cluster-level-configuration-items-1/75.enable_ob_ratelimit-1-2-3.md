enable_ob_ratelimit 
========================================
#docslug#/oceanbase-database/oceanbase-database/V4.0.0/enable_ob_ratelimit-1-2-3
`enable_ob_ratelimit` 用于在 RPC 连接中，启用区域之间的速率限制。


|      **属性**      |                                                 **描述**                                                 |
|------------------|--------------------------------------------------------------------------------------------------------|
| 参数类型             | 字符串                                                                                                    |
| 默认值              | False                                                                                                  |
| 取值范围             | <li> False   <li> True    |
| 是否重启 OBServer 生效 | 否                                                                                                      |
