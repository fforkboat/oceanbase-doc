enable_perf_event 
======================================
#docslug#/oceanbase-database/oceanbase-database/V4.0.0/enable_perf_event-1-2
enable_perf_event 用于设置是否开启性能事件的信息收集功能。


|      **属性**      |                                                 **描述**                                                 |
|------------------|--------------------------------------------------------------------------------------------------------|
| 参数类型             | 布尔类型                                                                                                   |
| 默认值              | True                                                                                                   |
| 取值范围             | <li> True   <li> False    |
| 是否重启 OBServer 生效 | 否                                                                                                      |
