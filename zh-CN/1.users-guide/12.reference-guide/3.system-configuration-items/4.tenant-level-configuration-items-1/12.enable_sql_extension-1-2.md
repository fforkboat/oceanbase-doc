enable_sql_extension 
=========================================
#docslug#/oceanbase-database/oceanbase-database/V4.0.0/enable_sql_extension-1-2
`enable_sql_extension` 用于设置是否开启租户的 SQL 扩展功能。


|      **属性**      |                                                    **描述**                                                     |
|------------------|---------------------------------------------------------------------------------------------------------------|
| 参数类型             | 布尔类型                                                                                                          |
| 默认值              | False                                                                                                         |
| 取值范围             | <li> True：开启   <li> False：不开启    |
| 是否重启 OBServer 生效 | 否                                                                                                             |
