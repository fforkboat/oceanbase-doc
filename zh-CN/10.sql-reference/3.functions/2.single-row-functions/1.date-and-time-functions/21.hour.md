HOUR 
=========================



声明 
-----------------------

```unknow
HOUR(time)
```



说明 
-----------------------

返回时间类型 `time` 的小时部分的数值。

示例 
-----------------------

```unknow
obclient> SELECT HOUR('10:05:03');
+------------------+
| HOUR('10:05:03') |
+------------------+
|               10 |
+------------------+
1 row in set (0.01 sec)

obclient> SELECT HOUR('272:59:59');
+-------------------+
| HOUR('272:59:59') |
+-------------------+
|               272 |
+-------------------+
1 row in set (0.01 sec)
```
