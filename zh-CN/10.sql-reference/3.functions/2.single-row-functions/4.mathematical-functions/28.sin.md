SIN 
========================



声明 
-----------------------

```unknow
SIN(X)
```



说明 
-----------------------

返回给定弧度 `X ` 的正弦值。

示例 
-----------------------

```unknow
obclient> SELECT SIN(PI());
+------------------------+
| SIN(PI())              |
+------------------------+
| 1.2246467991473532e-16 |
+------------------------+
1 row in set (0.00 sec)
```
