SECOND 
===========================



声明 
-----------------------

```unknow
SECOND(time)
```



说明 
-----------------------

返回时间 `time` 中的秒数。

示例 
-----------------------

```unknow
obclient> SELECT SECOND('18:20:17');
+--------------------+
| SECOND('18:20:17') |
+--------------------+
|                 17 |
+--------------------+
1 row in set (0.00 sec)
```
