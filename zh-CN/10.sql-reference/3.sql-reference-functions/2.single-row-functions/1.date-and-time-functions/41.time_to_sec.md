TIME_TO_SEC 
================================



声明 
-----------------------

```unknow
TIME_TO_SEC(time)
```



说明 
-----------------------

将时间类型参数 `time` 转换为秒。

示例 
-----------------------

```unknow
obclient> SELECT TIME_TO_SEC('22:23:00');
+-------------------------+
| TIME_TO_SEC('22:23:00') |
+-------------------------+
|                   80580 |
+-------------------------+
1 row in set (0.00 sec)

obclient> SELECT TIME_TO_SEC('00:39:38');
+-------------------------+
| TIME_TO_SEC('00:39:38') |
+-------------------------+
|                    2378 |
+-------------------------+
1 row in set (0.00 sec)
```
