DATABASE 
=============================



声明 
-----------------------

```unknow
DATABASE()
```



说明 
-----------------------

返回当前数据库名称，字符集为 utf8mb4。

如果没有通过 `USE` 语句指定数据库，则返回 `NULL`。

示例 
-----------------------

返回当前的数据库名称。

```unknow
obclient> SELECT DATABASE();
+------------+
| DATABASE() |
+------------+
| oceanbase  |
+------------+
1 row in set (0.00 sec)
```
