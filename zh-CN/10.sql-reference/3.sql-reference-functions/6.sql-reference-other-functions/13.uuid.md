UUID 
=========================



声明 
-----------------------

```unknow
uuid()
```



说明 
-----------------------

生成一个全局唯一 ID。

示例 
-----------------------

```javascript
obclient> SELECT UUID();
+--------------------------------------+
| UUID()                               |
+--------------------------------------+
| eef95d17-e944-11eb-824a-ca1fa9040697 |
+--------------------------------------+
1 row in set (0.00 sec)
```
