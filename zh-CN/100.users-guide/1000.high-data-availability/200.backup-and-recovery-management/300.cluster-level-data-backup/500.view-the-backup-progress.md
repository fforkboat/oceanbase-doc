# 查看备份进度

执行备份后，您可以查看备份进度。

## 通过视图查看数据备份进度以及备份任务历史

1. 使用 `root` 用户登录数据库的 `sys` 租户。

2. 进入 `oceanbase` 数据库。

   ```sql
   obclient> USE oceanbase;
   ```
