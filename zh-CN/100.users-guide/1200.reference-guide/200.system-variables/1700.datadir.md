datadir
============================

`datadir` 用于查看存储 SSTable 等数据的目录。

| **属性** |                                                   **描述**                                                   |
|--------|------------------------------------------------------------------------------------------------------------|
| 参数类型   | varchar                                                                                                    |
| 默认值    | /usr/local/mysql/data/                                                                                     |
| 取值范围   | N/A                                                                                                        |
| 生效范围   |  Global    |
| 是否可修改  | 否                                                                                                          |

> **说明**
>
> 该变量采用集群配置项 data_dir 的内容，该系统变量为只读变量不支持修改。关于 data_dir 的说明，请参考系统配置项 [data_dir](../300.system-configuration-items/4000.data_dir.md)
