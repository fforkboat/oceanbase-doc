sql_warnings
=================================

`sql_warnings` 用于在出现告警时控制单行 INSERT 语句是否生成信息字符串。

| **属性** |                                                        **描述**                                                         |
|--------|-----------------------------------------------------------------------------------------------------------------------|
| 参数类型   | bool                                                                                                                  |
| 默认值    | OFF                                                                                                                   |
| 取值范围   | <ul><li> ON：生成一个信息字符串 </li><li>OFF：不生成信息字符串</li></ul>    |
| 生效范围   | <ul><li>Global</li><li>Session</li></ul>               |
