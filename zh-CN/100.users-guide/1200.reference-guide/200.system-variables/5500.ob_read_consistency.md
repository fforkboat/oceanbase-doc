ob_read_consistency 
========================================

`ob_read_consistency` 用于设置读一致性级别。


|  **属性**  |                                                                                                **描述**                                                                                                 |
|----------|-------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------|
| 参数类型     | enum                                                                                                                                                                                                  |
| 默认值      | STRONG                                                                                                                                                                                                |
| 取值范围     | <ul><li> 0：空字符串 </li><li> 1：FROZEN </li><li>2：WEAK </li><li> 3：STRONG </li></ul>     |
| 生效范围     | <ul><li>Global</li><li>Session</li></ul>                                                                                               |
| 是否参与序列化  | 是                                                                                                                                                                                                     |
| 是否影响计划生成 | 是                                                                                                                                                                                                     |
