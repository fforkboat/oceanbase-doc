ob_enable_transformation 
=============================================

`ob_enable_transformation` 用于设置是否开启 SQL 优化器的改写功能。


|  **属性**  |                                                   **描述**                                                   |
|----------|------------------------------------------------------------------------------------------------------------|
| 参数类型     | bool                                                                                                       |
| 默认值      | 1                                                                                                          |
| 取值范围     | <ul><li> 0：不开启  </li><li> 1：开启</li></ul>        |
| 生效范围     | <ul><li>Global</li><li>Session</li></ul>    |
| 是否参与序列化  | 是                                                                                                          |
| 是否影响计划生成 | 是                                                                                                          |
