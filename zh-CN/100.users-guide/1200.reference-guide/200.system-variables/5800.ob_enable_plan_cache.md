ob_enable_plan_cache 
=========================================

`ob_enable_plan_cache` 用于设置是否打开 Plan Cache。


| **属性**  |                                                   **描述**                                                   |
|---------|------------------------------------------------------------------------------------------------------------|
| 参数类型    | bool                                                                                                       |
| 默认值     | 1                                                                                                          |
| 取值范围    | <ul><li> 0：关闭 </li><li>1：打开 </li></ul>        |
| 生效范围    | <ul><li>Global</li><li>Session</li></ul>    |
| 是否参与序列化 | 是                                                                                                          |
