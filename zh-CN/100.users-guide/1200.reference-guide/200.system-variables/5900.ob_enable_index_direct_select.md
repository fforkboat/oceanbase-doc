ob_enable_index_direct_select 
==================================================

`ob_enable_index_direct_select` 用于设置是否允许用户直接查询索引表。


|  **属性**  |                                                   **描述**                                                   |
|----------|------------------------------------------------------------------------------------------------------------|
| 参数类型     | bool                                                                                                       |
| 默认值      | 0                                                                                                          |
| 取值范围     | <ul><li>0：不允许 </li><li> 1：允许</li></ul>        |
| 生效范围     | <ul><li>Global</li><li>Session</li></ul>   |
| 是否参与序列化  | 是                                                                                                          |
| 是否影响计划生成 | 是                                                                                                          |
