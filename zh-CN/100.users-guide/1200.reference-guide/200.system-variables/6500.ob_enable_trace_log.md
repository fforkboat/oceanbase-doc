ob_enable_trace_log 
========================================

`ob_enable_trace_log` 用于设置是否使用 trace 日志。


| **属性**  |                                                   **描述**                                                   |
|---------|------------------------------------------------------------------------------------------------------------|
| 参数类型    | bool                                                                                                       |
| 默认值     | 0                                                                                                          |
| 取值范围    | <ul><li>0：不使用 </li><li>1：使用  </li></ul>      |
| 生效范围    | <ul><li>Global</li><li>Session</li></ul>    |
| 是否参与序列化 | 是                                                                                                          |
