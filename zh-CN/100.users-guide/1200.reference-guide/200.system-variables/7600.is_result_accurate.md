is_result_accurate 
=======================================

当查询带有 topk 提示时，`is_result_accurate` 表示结果是否正确。


| **属性**  |                                                 **描述**                                                 |
|---------|--------------------------------------------------------------------------------------------------------|
| 参数类型    | bool                                                                                                   |
| 默认值     | 1                                                                                                      |
| 取值范围    | <ul><li>0：不正确</li><li>1：正确 </li></ul>   |
| 生效范围    | Session                                                                                                |
| 是否参与序列化 | 是                                                                                                      |
