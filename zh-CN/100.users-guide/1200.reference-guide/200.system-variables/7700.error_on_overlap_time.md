error_on_overlap_time 
==========================================

`error_on_overlap_time` 是动态参数，用于设置夏令时重叠区时间发生歧义时是否作报错处理。


| **属性**  |                                                      **描述**                                                       |
|---------|-------------------------------------------------------------------------------------------------------------------|
| 参数类型    | bool                                                                                                              |
| 默认值     | OFF                                                                                                               |
| 取值范围    | <ul><li>ON：进行报错处理</li><li>OFF：不进行报错处理</li></ul>     |
| 生效范围    | <ul><li>Global</li><li>Session</li></ul>           |
| 是否参与序列化 | 是                                                                                                                 |
