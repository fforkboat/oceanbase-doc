foreign_key_checks 
=======================================



`foreign_key_checks` 用于设置是否在 DML 中检查外建约束。


| **属性**  |                                                     **描述**                                                     |
|---------|----------------------------------------------------------------------------------------------------------------|
| 参数类型    | bool                                                                                                           |
| 默认值     | 1                                                                                                              |
| 取值范围    | <ul><li> 0：不检查外键约束 </li><li>1：检查外键约束</li></ul>    |
| 生效范围    | <ul><li>Global</li><li>Session</li></ul>       |
| 是否参与序列化 | 是                                                                                                              |


> **说明**<br>
> 该变量设置后，仅对 DML 操作有效，DDL 操作不受影响。
