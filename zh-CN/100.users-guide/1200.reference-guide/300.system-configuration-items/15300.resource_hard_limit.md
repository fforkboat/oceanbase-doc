resource_hard_limit
========================================

`resource_hard_limit` 用于分配 unit 。

|      **属性**      |    **描述**    |
|------------------|--------------|
| 参数类型             | 整型           |
| 默认值              | 100          |
| 取值范围             | \[1, 10000\] |
| 是否重启 OBServer 生效 | 否            |

本配置项决定超卖的比例。默认 resource_hard_limit = 100 ，表示不超卖；如果设置的值大于 100，则表示超卖，数值越大，超卖比例越大。
