# query_response_time_range_base

`query_response_time_range_base` 用于设置 `QUERY_RESPONSE_TIME` 的间隔时间，仅仅在 `query_response_time_flush` 赋值影响后生效

| 属性     | 描述           |
| ------- | -------   |
| 参数类型    | 整数        |
| 默认值     | 10       |
| 取值范围    | \[2,10000\]            |
| 是否重启 OBServer 生效   | 否 |
