# role_change_timeout

<main id="notice" type='explain'>
  <h4>说明</h4>
  <p>该配置项从V3.1.5_CE_HF2 版本引入。</p>
</main>

## 功能描述

`role_change_timeout` 用于设置副本 Leader 上任/卸任的超时时间，属于集群级配置项。

## 属性说明

| **属性** | **描述** |
| ----- | ----- |
| 参数类型 | time |
| 默认值 | 10s |
| 取值范围 | (0s, +∞) |
| 是否重启 OBServer 节点生效 | 否 |

## 配置示例

```sql
ALTER SYSTEM SET role_change_timeout='10s';
```
