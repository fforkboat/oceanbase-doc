clog_disk_usage_limit_percentage 
=====================================================

`clog_disk_usage_limit_percentage` 用于设置事务日志可用磁盘空间的最大百分比。


|      **属性**      |   **描述**    |
|------------------|-------------|
| 参数类型             | 整型          |
| 默认值              | 95          |
| 取值范围             | \[80, 100\] |
| 是否重启 OBServer 生效 | 否           |


> **说明**<br>
> 当事务日志对应的磁盘空间的使用量达到该配置项设置的值后，磁盘将停止写入，且无法再接收日志。
