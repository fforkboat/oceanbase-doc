freeze_trigger_percentage 
==============================================

`freeze_trigger_percentage` 用于设置触发全局冻结的租户使用内存阈值。


|      **属性**      |  **描述**   |
|------------------|-----------|
| 参数类型             | 整型        |
| 默认值              | 50      |
| 取值范围             | \[1, 99\] |
| 是否重启 OBServer 生效 | 否         |



如果开启了自动触发全局冻结，则当租户使用的内存达到该阈值时，会触发全局冻结。自动触发全局冻结开关的设置请参见 [enable_global_freeze_trigger](../300.system-configuration-items/6100.enable_global_freeze_trigger.md)。
