# __min_full_resource_pool_memory

`__min_full_resource_pool_memory` 用于控制全功能型资源池的最小的内存规格。

<main id="notice" type='explain'>
  <h4>说明</h4>
  <p>资源池默认为全功能型资源池，无需特别关注。</p>
</main>

| **属性** | **描述** |
| --- | --- |
| 参数类型 | 整型 |
| 默认值 | 5368709120 |
| 取值范围 | [268435456, +∞) |
| 是否重启 OBServer 节点生效 | 否 |
