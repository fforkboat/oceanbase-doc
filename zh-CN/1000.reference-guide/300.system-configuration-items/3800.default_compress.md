default_compress 
=====================================

default_compress 用于设置在 Oracle 模式下，建表时使用的默认压缩策略。


|      **属性**      |                                                                                                                     **描述**                                                                                                                     |
|------------------|------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------|
| 参数类型             | 字符串类型                                                                                                                                                                                                                                          |
| 默认值              | OLTP                                                                                                                                                                                                                                        |
| 取值范围             | <ul><li>ARCHIVE</li><li>BASIC</li><li>NOCOMPRESS</li><li>OLTP</li><li>QUERY</li></ul>    |
| 是否重启 OBServer 生效 | 否                                                                                                                                                                                                                                              |
