default_row_format 
=======================================

default_row_format 用于设置在 MySQL 模式下，建表时使用的默认行格式。


|      **属性**      |                                                                                                  **描述**                                                                                                   |
|------------------|-----------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------|
| 参数类型             | 字符串                                                                                                                                                                                                       |
| 默认值              | COMPACT                                                                                                                                                                                                   |
| 取值范围             |  <ul><li>COMPACT</li><li>COMPRESSED</li><li>DYNAMIC</li><li>REDUNDANT</li></ul>    |
| 是否重启 OBServer 生效 | 否                                                                                                                                                                                                         |
