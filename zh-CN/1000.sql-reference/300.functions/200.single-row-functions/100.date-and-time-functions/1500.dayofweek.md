DAYOFWEEK 
==============================



声明 
-----------------------

```unknow
DAYOFWEEK(date)
```



说明 
-----------------------

返回日期或时间日期 `date` 在一周中的位置，从周日开始计算。

示例 
-----------------------

```unknow
obclient> SELECT DAYOFWEEK('2021-07-15');
+-------------------------+
| DAYOFWEEK('2021-07-15') |
+-------------------------+
|                       5 |
+-------------------------+
1 row in set (0.01 sec)
```
