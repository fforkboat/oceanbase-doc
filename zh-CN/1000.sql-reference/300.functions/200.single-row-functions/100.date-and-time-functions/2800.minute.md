MINUTE 
===========================



声明 
-----------------------

```unknow
MINUTE(time)
```



说明 
-----------------------

返回时间 `time` 中的分钟数。

示例 
-----------------------

```unknow
obclient> SELECT MINUTE('18:20:17');
+--------------------+
| MINUTE('18:20:17') |
+--------------------+
|                 20 |
+--------------------+
1 row in set (0.00 sec)
```
