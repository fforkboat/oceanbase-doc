QUARTER 
============================



声明 
-----------------------

```unknow
QUARTER(date)
```



说明 
-----------------------

返回指定日期的季度值，取值范围为 \[1,4\]。

示例 
-----------------------

```unknow
obclient> SELECT QUARTER(NOW()), NOW();
+----------------+---------------------+
| QUARTER(NOW()) | NOW()               |
+----------------+---------------------+
|              3 | 2021-07-12 19:45:01 |
+----------------+---------------------+
1 row in set (0.00 sec)
```
