SEC_TO_TIME 
================================



声明 
-----------------------

```unknow
SEC_TO_TIME(seconds)
```



说明 
-----------------------

返回 `seconds` 转换为的时间类型。结果的范围仅限于 `TIME` 数据类型的范围。

示例 
-----------------------

```unknow
obclient> SELECT SEC_TO_TIME(2378);
+-------------------+
| SEC_TO_TIME(2378) |
+-------------------+
| 00:39:38          |
+-------------------+
1 row in set (0.00 sec)

obclient> SELECT SEC_TO_TIME(2378) + 0;
+-----------------------+
| SEC_TO_TIME(2378) + 0 |
+-----------------------+
|                  3938 |
+-----------------------+
1 row in set (0.00 sec)
```
