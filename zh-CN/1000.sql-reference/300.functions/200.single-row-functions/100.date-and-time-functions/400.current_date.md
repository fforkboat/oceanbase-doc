CURRENT_DATE 
=================================



声明 
-----------------------

```unknow
CURRENT_DATE()
CURRENT_DATE
```



说明 
-----------------------

返回当前日期，不含时间部分。该函数是 [CURDATE](../100.date-and-time-functions/300.curdate.md) 的别名。

示例 
-----------------------

```javascript
obclient> SELECT CURRENT_DATE;
+--------------+
| CURRENT_DATE |
+--------------+
| 2021-08-23   |
+--------------+
1 row in set (0.00 sec)

obclient> SELECT CURRENT_DATE();
+----------------+
| CURRENT_DATE() |
+----------------+
| 2021-08-23     |
+----------------+
1 row in set (0.00 sec)
```
