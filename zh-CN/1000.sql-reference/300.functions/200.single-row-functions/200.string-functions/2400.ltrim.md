LTRIM 
==========================



声明 
-----------------------

```javascript
LTRIM(str)
```



说明 
-----------------------

删除字符串左侧的空格。

示例 
-----------------------

```unknow
obclient> SELECT LTRIM('    obclient    ') AS ltrim;
+--------------+
| ltrim        |
+--------------+
| obclient     |
+--------------+
1 row in set (0.01 sec)
```
