OCT 
========================



声明 
-----------------------

```unknow
OCT(N)
```



描述 
-----------------------

返回 `N` 的八进制值，该函数等价于 `CONV(N,10,8)`。

如果 `N` 为 `NULL`，则该函数返回 `NULL`。

示例 
-----------------------

```unknow
obclient> SELECT OCT(12);
+---------+
| OCT(12) |
+---------+
| 14      |
+---------+
1 row in set (0.00 sec)
```
