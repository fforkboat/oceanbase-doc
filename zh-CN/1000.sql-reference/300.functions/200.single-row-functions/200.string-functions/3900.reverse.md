REVERSE 
============================



声明 
-----------------------

```unknow
REVERSE(str)
```



说明 
-----------------------

返回字符串 `str` 的倒序排列。

该函数是多字节安全的。

示例 
-----------------------

```unknow
obclient> SELECT REVERSE('oceanbase');
+----------------------+
| REVERSE('oceanbase') |
+----------------------+
| esabnaeco            |
+----------------------+
1 row in set (0.00 sec)

obclient> SELECT REVERSE(1000);
+---------------+
| REVERSE(1000) |
+---------------+
| 0001          |
+---------------+
1 row in set (0.00 sec)
```
