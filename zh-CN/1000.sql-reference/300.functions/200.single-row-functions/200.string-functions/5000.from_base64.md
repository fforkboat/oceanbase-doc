FROM_BASE64 
================================



声明 
-----------------------

```sql
FROM_BASE64(str)
```



说明 
-----------------------

获取使用 base-64 编码规则编码的字符串，并将解码结果作为二进制字符串返回。

如果参数为 `NULL` 或不是有效的 base-64 字符串，则结果为 `NULL`。 

有关 base-64 编码的详细信息，请参见 [TO_BASE64](../2.string-functions/49.to_base64.md) 。

示例 
-----------------------

```sql
obclient> SELECT FROM_BASE64('MTIzNDU=');
+-------------------------+
| FROM_BASE64('MTIzNDU=') |
+-------------------------+
| 12345                   |
+-------------------------+
1 row in set
```
