COS 
========================



声明 
-----------------------

```unknow
COS(X)
```



说明 
-----------------------

返回给定弧度 `X` 的余弦值。

示例 
-----------------------

```unknow
obclient> SELECT COS(PI());
+-----------+
| COS(PI()) |
+-----------+
|        -1 |
+-----------+
1 row in set (0.01 sec)
```
