LOG2 
=========================



声明 
-----------------------

```unknow
LOG2(X)
```



说明 
-----------------------

返回以 2 为底的 `X` 的指数。

示例 
-----------------------

```unknow
obclient> SELECT LOG2(65536);
+-------------+
| LOG2(65536) |
+-------------+
|          16 |
+-------------+
1 row in set (0.00 sec)

obclient> SELECT LOG2(-65536);
+--------------+
| LOG2(-65536) |
+--------------+
|         NULL |
+--------------+
1 row in set, 1 warning (0.00 sec)
```
