INET_NTOA 
==============================



声明 
-----------------------

```unknow
INET_NTOA(expr)
```



说明 
-----------------------

将按照网络字节顺序表示的 IPv4 网络地址的数值，返回为使用"."点隔的 IP 地址字符串并作为连接字符集中的字符串。如果参数无效，则返回 `NULL`。

示例 
-----------------------

```unknow
obclient> SELECT INET_NTOA(16843009);
+---------------------+
| INET_NTOA(16843009) |
+---------------------+
| 1.1.1.1             |
+---------------------+
1 row in set
```
