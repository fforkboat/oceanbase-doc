DAYOFYEAR 
==============================



声明 
-----------------------

```unknow
DAYOFYEAR(date)
```



说明 
-----------------------

返回日期或时间日期 `date` 在一年中的位置。

示例 
-----------------------

```unknow
obclient> SELECT DAYOFYEAR('2021-07-15');
+-------------------------+
| DAYOFYEAR('2021-07-15') |
+-------------------------+
|                     196 |
+-------------------------+
1 row in set (0.00 sec)
```
