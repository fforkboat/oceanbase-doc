YEAR 
=========================



声明 
-----------------------

```javascript
YEAR(date)
```



说明 
-----------------------

返回 `date` 值的年份信息。

示例 
-----------------------

```javascript
obclient> SELECT YEAR('1987-01-01');
+--------------------+
| YEAR('1987-01-01') |
+--------------------+
|               1987 |
+--------------------+
1 row in set (0.00 sec)
```
