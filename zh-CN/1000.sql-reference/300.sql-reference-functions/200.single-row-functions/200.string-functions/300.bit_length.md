BIT_LENGTH 
===============================



声明 
-----------------------

```unknow
BIT_LENGTH(str)
```



说明 
-----------------------

返回字符串 `str` 的比特位长度。

示例 
-----------------------

```unknow
obclient> SELECT BIT_LENGTH('oceanbase');
+-------------------------+
| BIT_LENGTH('oceanbase') |
+-------------------------+
|                      72 |
+-------------------------+
1 row in set (0.00 sec)
```
