POSITION 
=============================



声明 
-----------------------

```unknow
POSITION(substr IN str)
```



说明 
-----------------------

回字符串 `str` 中子字符串 `substr` 的第一个出现位置。

示例 
-----------------------

```unknow
obclient> SELECT POSITION('bar' IN 'foobarbar');
+--------------------------------+
| POSITION('bar' IN 'foobarbar') |
+--------------------------------+
|                              4 |
+--------------------------------+
1 row in set (0.01 sec)
```
