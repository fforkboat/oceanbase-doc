TAN 
========================



声明 
-----------------------

```unknow
TAN(X)
```



说明 
-----------------------

返回给定弧度 X 的正切值。

示例 
-----------------------

```unknow
obclient> SELECT TAN(PI());
+-------------------------+
| TAN(PI())               |
+-------------------------+
| -1.2246467991473532e-16 |
+-------------------------+
1 row in set (0.00 sec)
```
