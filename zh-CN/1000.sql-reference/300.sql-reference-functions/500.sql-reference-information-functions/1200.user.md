USER 
=========================



声明 
-----------------------

```unknow
USER()
```



说明 
-----------------------

返回当前的用户名与主机名，返回值使用的字符集为 utf8mb4。

示例 
-----------------------

```unknow
obclient> SELECT USER();
+---------------------+
| USER()              |
+---------------------+
| root@xxx.xx.xxx.xxx |
+---------------------+
1 row in set (0.00 sec)
```
