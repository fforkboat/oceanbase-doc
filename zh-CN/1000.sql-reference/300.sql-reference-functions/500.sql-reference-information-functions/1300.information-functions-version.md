VERSION 
============================



声明 
-----------------------

```unknow
VERSION()
```



说明 
-----------------------

返回当前 OceanBase 数据库的版本信息，字符集为 utf8mb4。

示例 
-----------------------

```unknow
obclient> SELECT VERSION();
+-----------+
| VERSION() |
+-----------+
| 3.2.0     |
+-----------+
1 row in set (0.01 sec)
```
