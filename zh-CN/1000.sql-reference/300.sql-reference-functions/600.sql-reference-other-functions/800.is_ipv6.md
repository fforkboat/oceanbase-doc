IS_IPV6 
============================



声明 
-----------------------

```unknow
IS_IPV6(expr)
```



说明 
-----------------------

如果参数 `expr` 是有效的 IPv6 地址字符串，则返回 1，否则返回 0。此函数不会将 IPv4 地址视为有效的 IPv6 地址。

对于指定的参数，如果 `IS_IPV6()` 返回 1，则 `INET6_ATON()` 返回非 `NULL`。

示例 
-----------------------

```unknow
obclient> SELECT IS_IPV6('10.0.5.9'), IS_IPV6('::1');
+---------------------+----------------+
| IS_IPV6('10.0.5.9') | IS_IPV6('::1') |
+---------------------+----------------+
|                   0 |              1 |
+---------------------+----------------+
1 row in set
```
