tableapi_transport_compress_func 
=====================================================

`tableapi_transport_compress_func` 用于设置 tableAPI 查询结果传输使用的压缩算法。


|      **属性**      |                                                                                                                                                **描述**                                                                                                                                                |
|------------------|------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------|
| 参数类型             | 字符串类型                                                                                                                                                                                                                                                                                                |
| 默认值              | none                                                                                                                                                                                                                                                                                                 |
| 取值范围             | <li> none   </li><li> lz4_1.0   </li><li> snappy_1.0   </li><li> zlib_1.0   </li><li> zstd_1.0   </li><li> zstd 1.3.8    |
| 是否重启 OBServer 生效 | 否                                                                                                                                                                                                                                                                                                    |
