gc_wait_archive 
====================================

`gc_wait_archive` 用于设置分区 GC 时是否需要等待分区上所有日志归档完成。


|      **属性**      |                                                 **描述**                                                 |
|------------------|--------------------------------------------------------------------------------------------------------|
| 参数类型             | 布尔类型                                                                                                   |
| 默认值              | True                                                                                                   |
| 取值范围             | <li> False   </li><li> True    |
| 是否重启 OBServer 生效 | 否                                                                                                      |
