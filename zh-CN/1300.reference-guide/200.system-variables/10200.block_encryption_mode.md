block_encryption_mode 
==========================================

block_encryption_mode 用于指定在函数 aes_encrypt 和 aes_decrypt 中使用的加密算法。


| **属性**  |                                                                                                                                                                                                                                                                                                                                                                                                                                                                       **描述**                                                                                                                                                                                                                                                                                                                                                                                                                                                                       |
|---------|----------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------|
| 参数类型    | enum                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                               |
| 默认值     | aes-128-ecb                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                        |
| 取值范围    | <ul><li>aes-128-ecb</li><li>aes-192-ecb</li><li>aes-256-ecb</li><li>aes-128-cbc</li><li>aes-192-cbc</li><li>aes-256-cbc</li><li>aes-128-cfb1</li><li>aes-192-cfb1</li><li>aes-256-cfb1</li><li>aes-128-cfb8</li><li>aes-192-cfb8</li><li>aes-256-cfb8</li><li>aes-128-cfb128</li><li>aes-192-cfb128</li><li>aes-256-cfb128</li><li>aes-128-ofb</li><li>aes-192-ofb</li><li>aes-256-ofb</li></ul>      |
| 生效范围    |<ul><li>Global</li><li>Session</li></ul>                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                             |
| 是否参与序列化 | 是                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                  |
