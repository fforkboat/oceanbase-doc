# PERIOD_ADD

## 声明

```sql
PERIOD_ADD(P,N)
```

## 说明

向时间段 `P` 添加 `N` 个月。其中 `P` 的格式为 `YYMM` 或 `YYYYMM`。

返回值的格式为 `YYYYMM`。

>**说明**
>
>参数 `P` 不能是 `DATE` 类型，即非日期值。

## 示例

```sql
obclient> SELECT PERIOD_ADD(200801,2);
+----------------------+
| PERIOD_ADD(200801,2) |
+----------------------+
|               200803 |
+----------------------+
1 row in set
```
