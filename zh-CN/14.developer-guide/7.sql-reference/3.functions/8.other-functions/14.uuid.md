# UUID

## 声明

```sql
uuid()
```

## 说明

生成一个全局唯一 ID。

## 示例

```sql
obclient> SELECT UUID();
+--------------------------------------+
| UUID()                               |
+--------------------------------------+
| eef95d17-e944-11eb-824a-ca1fa904**** |
+--------------------------------------+
1 row in set
```
