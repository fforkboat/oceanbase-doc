# DROP RESOURCE POOL

## 描述

该语句用来删除资源池。

>**注意**
>
>已经分配给租户的资源池不能删除。

## 语法

```sql
DROP RESOURCE POOL pool_name; 
```

## 参数解释

|  **参数**   |   **描述**   |
|-----------|------------|
| pool_name | 要删除的资源池名称。 |

## 示例

删除资源池 `pool1`。

```sql
obclient> DROP RESOURCE POOL pool1;
Query OK, 0 rows affected
```
