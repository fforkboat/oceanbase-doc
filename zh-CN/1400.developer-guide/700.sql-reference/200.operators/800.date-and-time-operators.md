# 日期时间运算符

OceanBase 数据库支持日期时间运算符，其功能等同于指定的时间日期函数。

OceanBase 数据库当前版本所支持的日期时间运算符的详细信息如下表所示。

| 运算符 | 操作数 |                                                    含义                                                    |
|-----|-----|----------------------------------------------------------------------------------------------------------|
| `+` | 二元  | 将日期和时间相加，等同于 `DATE_ADD` 函数。 请参见 [DATE_ADD](../300.functions/200.single-row-functions/100.date-and-time-functions/900.date_add.md)。 |
| `-` | 二元  | 将日期和时间相减，等同于 `DATE_SUB` 函数。 请参见 [DATE_SUB](../300.functions/200.single-row-functions/100.date-and-time-functions/1100.date_sub.md)。 |

示例如下：

```sql
obclient> SELECT '2008-12-31 23:59:59' + INTERVAL 1 SECOND;
+-------------------------------------------+
| '2008-12-31 23:59:59' + INTERVAL 1 SECOND |
+-------------------------------------------+
| 2009-01-01 00:00:00                       |
+-------------------------------------------+
1 row in set
```
