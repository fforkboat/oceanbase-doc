# TIME

## 声明

```sql
TIME(datetime)
```

## 说明

以 `TIME` 类型返回 `datetime` 的时间信息。

## 示例

```sql
obclient> SELECT TIME('2003-12-31 01:02:03');
+-----------------------------+
| TIME('2003-12-31 01:02:03') |
+-----------------------------+
| 01:02:03.000000             |
+-----------------------------+
1 row in set
```
