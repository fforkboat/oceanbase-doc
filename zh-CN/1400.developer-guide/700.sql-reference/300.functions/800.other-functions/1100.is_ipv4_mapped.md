# IS_IPV4_MAPPED

## 声明

```sql
IS_IPV4_MAPPED(expr)
```

## 说明

将以数字形式表示的 IPv6 地址作为二进制字符串，如果参数是有效的 IPv4 映射的 IPv6 地址，则返回 1，否则返回 0。IPv4 映射地址的格式为 `::ffff:ipv4_address`。

## 示例

```sql
obclient> SELECT IS_IPV4_MAPPED(INET6_ATON('::10.0.5.9'));
+------------------------------------------------+
|   IS_IPV4_MAPPED(INET6_ATON('::10.0.5.9'))|
+------------------------------------------------+
|                                                                     0 |
+------------------------------------------------+
1 row in set
```
