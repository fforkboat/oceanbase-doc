查看租户信息 
===========================

您可以通过 SQL 语句查看租户的信息。

通过 SQL 语句查看 
--------------------------------

通过 OBClient 中登录到 OceanBase 数据库的 `sys` 租户，并执行以下语句，可以查看当前集群的租户信息。如果是普通租户，则只能查看当前租户信息。

```sql
obclient> SELECT * FROM oceanbase.gv$tenant;
```



此外，还可以执行以下语句快速查看当前租户名。

```sql
obclient> SHOW TENANT;
```
