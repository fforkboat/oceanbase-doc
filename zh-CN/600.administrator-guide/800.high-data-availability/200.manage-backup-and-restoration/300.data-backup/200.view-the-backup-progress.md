查看备份进度 
===========================

执行备份后，您可以查看备份进度。

通过视图查看数据备份进度以及备份任务历史 
-----------------------------------------

1. 使用 `root` 用户登录数据库的 `sys` 租户。

   

2. 进入 `oceanbase` 数据库。

   ```sql
   obclient> USE oceanbase;
   ```

   

3. 进行以下操作：

   * 查看备份进度

     ```sql
     obclient> SELECT * FROM CDB_OB_BACKUP_PROGRESS;
     ```

     
   
   * 查看备份历史

     ```sql
     obclient> SELECT * FROM CDB_OB_BACKUP_SET_DETAILS;
     ```

     
   

   

   更多 `CDB_OB_BACKUP_PROGRESS` 和 `CDB_OB_BACKUP_SET_DETAILS` 视图的说明，请参见 [备份恢复相关视图](../600.backup-and-restoration-related-views.md)。
