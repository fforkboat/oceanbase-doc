# 查看租户的资源配置

您可以根据需要，查看租户的资源配置，包括资源池中资源单元个数和资源单元的具体配置。

## 操作步骤

1. 使用 `root` 用户登录集群的 `sys` 租户。

2. 进入 oceanbase 数据库。

   ```sql
   obclient> USE oceanbase;
   ```

3. 执行以下语句，可以查看集群内某个租户所使用的资源信息。

   ```sql
   SELECT * FROM oceanbase.GV$OB_UNITS WHERE tenant_id='1002'\G
   ```

   查询结果如下：

