MINUTE 
===========================
#docslug#/oceanbase-database/oceanbase-database/V4.0.0/MINUTE


声明 
-----------------------

```sql
MINUTE(time)
```



说明 
-----------------------

返回时间 `time` 中的分钟数。

示例 
-----------------------

```sql
obclient> SELECT MINUTE('18:20:17');
+--------------------+
| MINUTE('18:20:17') |
+--------------------+
|                 20 |
+--------------------+
1 row in set 
```
