MONTH 
==========================
#docslug#/oceanbase-database/oceanbase-database/V4.0.0/MONTH


声明 
-----------------------

```javascript
MONTH(date)
```



说明 
-----------------------

返回 `date` 的月份信息。

示例 
-----------------------

```javascript
obclient> SELECT MONTH('2008-02-03');
+---------------------+
| MONTH('2008-02-03') |
+---------------------+
|                   2 |
+---------------------+
1 row in set 
```
