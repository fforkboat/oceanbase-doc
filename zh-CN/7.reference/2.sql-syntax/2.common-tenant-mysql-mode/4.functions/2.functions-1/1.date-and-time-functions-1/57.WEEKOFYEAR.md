WEEKOFYEAR 
===============================
#docslug#/oceanbase-database/oceanbase-database/V4.0.0/WEEKOFYEAR


声明 
-----------------------

```sql
WEEKOFYEAR(date)
```



说明 
-----------------------

返回日期或时间日期类型 `date` 所指定的周在一年中的位置。

示例 
-----------------------

```sql
obclient> SELECT WEEKOFYEAR('2020-07-16');
+--------------------------+
| WEEKOFYEAR('2020-07-16') |
+--------------------------+
|                       29 |
+--------------------------+
1 row in set 
```
