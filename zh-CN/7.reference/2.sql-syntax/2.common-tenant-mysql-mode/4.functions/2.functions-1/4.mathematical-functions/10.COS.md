COS 
========================
#docslug#/oceanbase-database/oceanbase-database/V4.0.0/COS


声明 
-----------------------

```sql
COS(X)
```



说明 
-----------------------

返回给定弧度 `X` 的余弦值。

示例 
-----------------------

```sql
obclient> SELECT COS(PI());
+-----------+
| COS(PI()) |
+-----------+
|        -1 |
+-----------+
1 row in set 
```
