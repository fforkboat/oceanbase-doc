PURGE RECYCLEBIN 
=====================================
#docslug#/oceanbase-database/oceanbase-database/V4.0.0/purge-recyclebin


描述 
-----------------------

该语句用于清空回收站。
>**注意**
>
>`PURGE RECYCLEBIN` 会清除回收站中的所有数据库对象，只有 root 用户才能够执行这条语句，操作时请谨慎。

语法 
-----------------------

```sql
 PURGE RECYCLEBIN;
```



示例 
-----------------------

清空回收站。

```sql
obclient> PURGE RECYCLEBIN;
Query OK, 0 rows affected 
```
