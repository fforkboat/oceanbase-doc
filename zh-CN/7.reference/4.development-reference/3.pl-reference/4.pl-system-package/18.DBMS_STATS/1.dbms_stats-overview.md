# DBMS_STATS 概述 

`DBMS_STATS` 系统包用于查看和修改优化器为数据库对象收集的统计信息。

## DBMS_STATS 使用说明 

OceanBase 数据库当前版本所支持的 `DBMS_STATS` 系统包主要支持以下常规操作：

* 收集优化器统计信息 

* 设置列、表和索引的相关信息 

* 删除统计信息

* 锁定和解锁统计信息
