performance_schema 
=======================================
#docslug#/oceanbase-database/oceanbase-database/V4.0.0/performance_schema-1-2-3
performance_schema 用于向客户端声明是否支持 performance 信息的查询。


|   属性   |                                                    描述                                                     |
|--------|-----------------------------------------------------------------------------------------------------------|
| 参数类型   | bool                                                                                                      |
| 默认值    | OFF                                                                                                       |
| 取值范围   | <li> ON：支持   <li> OFF：不支持    |
| 生效范围   | GLOBAL                                                                                                    |
| 是否重启生效 | 否                                                                                                         |
