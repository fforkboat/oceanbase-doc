validate_password_mixed_case_count 
=======================================================
#docslug#/oceanbase-database/oceanbase-database/V4.0.0/validate_password_mixed_case_count-1-2
validate_password_mixed_case_count 用于设置用户密码至少包含的大写字母个数以及至少包含的小写字母个数。


| **属性**  |      **描述**      |
|---------|------------------|
| 参数类型    | uint             |
| 默认值     | 0                |
| 取值范围    | \[0,2147483647\] |
| 生效范围    | GLOBAL           |
| 是否参与序列化 | 是                |
