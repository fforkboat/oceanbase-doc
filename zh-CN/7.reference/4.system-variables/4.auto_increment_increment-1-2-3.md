auto_increment_increment 
=============================================
#docslug#/oceanbase-database/oceanbase-database/V4.0.0/auto_increment_increment-1-2-3
auto_increment_increment 用于设置自增步长，仅用于 MySQL 客户端登录。


| **属性**  |                                                   **描述**                                                   |
|---------|------------------------------------------------------------------------------------------------------------|
| 参数类型    | uint                                                                                                       |
| 默认值     | 1                                                                                                          |
| 取值范围    | \[1, 65535\]                                                                                               |
| 生效范围    | <li> GLOBAL   <li> SESSION    |
| 是否参与序列化 | 是                                                                                                          |
