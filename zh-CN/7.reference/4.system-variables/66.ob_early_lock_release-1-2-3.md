ob_early_lock_release 
==========================================
#docslug#/oceanbase-database/oceanbase-database/V4.0.0/ob_early_lock_release-1-2-3
`ob_early_lock_release` 用于控制事务是否开启提前解锁（ELR）功能。

> **说明**
> 
> 该变量已不再使用。


| **属性**  |                                                 **描述**                                                 |
|---------|--------------------------------------------------------------------------------------------------------|
| 参数类型    | bool                                                                                                   |
| 默认值     | 0                                                                                                      |
| 取值范围    | <li> 0：不开启   <li> 1：开启    |
| 生效范围    | GLOBAL                                                                                                 |
| 是否参与序列化 | 是                                                                                                      |
