ob_enable_hash_group_by 
============================================
#docslug#/oceanbase-database/oceanbase-database/V4.0.0/ob_enable_hash_group_by-1-2-3
ob_enable_hash_group_by 用于设置是否打开 Hash Group by 的路径。


|  **属性**  |                                                   **描述**                                                   |
|----------|------------------------------------------------------------------------------------------------------------|
| 参数类型     | bool                                                                                                       |
| 默认值      | 1                                                                                                          |
| 取值范围     | <li> 0：不允许   <li> 1：允许        |
| 生效范围     | <li> GLOBAL   <li> SESSION    |
| 是否参与序列化  | 是                                                                                                          |
| 是否影响计划生成 | 是                                                                                                          |
