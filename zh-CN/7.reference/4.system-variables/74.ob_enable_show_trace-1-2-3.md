# ob_enable_show_trace
#docslug#/oceanbase-database/oceanbase-database/V4.0.0/ob_enable_show_trace-1-2-3
ob_enable_show_trace 用于设置是否开启 show trace 功能。

| **属性**  |                                                   **描述**                                                   |
|---------|------------------------------------------------------------------------------------------------------------|
| 参数类型    | bool                                                                                                       |
| 默认值     | 0                                                                                                          |
| 取值范围    | <li> 0：不开启   <li> 1：开启        |
| 生效范围    | <li> GLOBAL   <li> SESSION    |
| 是否参与序列化 | 是                                                                                                          |
