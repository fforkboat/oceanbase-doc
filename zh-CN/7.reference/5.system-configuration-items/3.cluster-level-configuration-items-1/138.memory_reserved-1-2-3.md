memory_reserved
====================================
#docslug#/oceanbase-database/oceanbase-database/V4.0.0/memory_reserved-1-2-3
memory_reserved 用于设置系统预留给 KVCache 的内存容量。


|      **属性**      |   **描述**   |
|------------------|------------|
| 参数类型             | 容量单位       |
| 默认值              | 500M       |
| 取值范围             | \[10M, +∞) |
| 是否重启 OBServer 生效 | 否          |
