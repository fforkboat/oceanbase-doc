migration_disable_time 
===========================================
#docslug#/oceanbase-database/oceanbase-database/V4.0.0/migration_disable_time-1-2-3
migration_disable_time 用于设置因磁盘满等原因导致某个节点数据迁入失败时的暂停迁入时长。


|      **属性**      |  **描述**   |
|------------------|-----------|
| 参数类型             | 时间类型      |
| 默认值              | 3600s     |
| 取值范围             | \[1s, +∞) |
| 是否重启 OBServer 生效 | 否         |
