row_compaction_update_limit 
================================================
#docslug#/oceanbase-database/oceanbase-database/V4.0.0/row_compaction_update_limit-1-2-3
row_compaction_update_limit 用于设置触发内存中行内数据合并的修改次数。


|      **属性**      |   **描述**    |
|------------------|-------------|
| 参数类型             | 整型          |
| 默认值              | 6           |
| 取值范围             | \[1, 6400\] |
| 是否重启 OBServer 生效 | 否           |
