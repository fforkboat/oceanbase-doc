rpc_port 
=============================
#docslug#/oceanbase-database/oceanbase-database/V4.0.0/rpc_port-1-2-3
rpc_port 用于设置远程访问的协议端口号。


|      **属性**      |     **描述**      |
|------------------|-----------------|
| 参数类型             | 整型              |
| 默认值              | 2882            |
| 取值范围             | \[1025, 65535\] |
| 是否重启 OBServer 生效 | 否               |
