server_cpu_quota_min 
=========================================
#docslug#/oceanbase-database/oceanbase-database/V4.0.0/server_cpu_quota_min-1-2-3
server_cpu_quota_min 用于设置系统可以使用的最小 CPU 配额，系统会自动预留。

|      **属性**      |  **描述**   |
|------------------|-----------|
| 参数类型             | 双精度浮点数    |
| 默认值               | 1              |
| 取值范围             | \[1, 16\]       |
| 是否重启 OBServer 生效 | 是             |
