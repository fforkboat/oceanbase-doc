system_memory
==================================
#docslug#/oceanbase-database/oceanbase-database/V4.0.0/system_memory-1-2-3
`system_memory` 用于设置系统预留给租户 ID 为 `500` 的租户的内存容量。


|      **属性**      |  **描述**   |
|------------------|-----------|
| 参数类型             | 容量单位      |
| 默认值              | 0M       |
| 取值范围             | \[0M, +∞) |
| 是否重启 OBServer 生效 | 否         |


> **注意**
>
> 系统预留内存不能分配给普通租户使用。
