balancer_idle_time 
=======================================
#docslug#/oceanbase-database/oceanbase-database/V4.0.0/balancer_idle_time-1-2-3
balancer_idle_time 用于设置负载均衡等后台任务线程空闲时的唤醒间隔时间。


|      **属性**      |   **描述**   |
|------------------|------------|
| 参数类型             | 时间类型       |
| 默认值              | 5m，表示 5 分钟 |
| 取值范围             | \[10s, +∞) |
| 是否重启 OBServer 生效 | 否          |
