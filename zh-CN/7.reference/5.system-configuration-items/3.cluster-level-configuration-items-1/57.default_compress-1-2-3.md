default_compress 
=====================================
#docslug#/oceanbase-database/oceanbase-database/V4.0.0/default_compress-1-2-3
default_compress 用于设置在 Oracle 模式下，建表时使用的默认压缩策略。


|      **属性**      |                                                                                                                     **描述**                                                                                                                     |
|------------------|------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------|
| 参数类型             | 字符串类型                                                                                                                                                                                                                                          |
| 默认值              | ARCHIVE                                                                                                                                                                                                                                        |
| 取值范围             | <li> NOCOMPRESS   <li> BASIC   <li> OLTP   <li> QUERY   <li> ARCHIVE    |
| 是否重启 OBServer 生效 | 否                                                                                                                                                                                                                                              |
