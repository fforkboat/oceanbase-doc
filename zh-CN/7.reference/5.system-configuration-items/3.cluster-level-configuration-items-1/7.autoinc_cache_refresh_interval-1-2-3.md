autoinc_cache_refresh_interval 
===================================================
#docslug#/oceanbase-database/oceanbase-database/V4.0.0/autoinc_cache_refresh_interval-1-2-3
autoinc_cache_refresh_interval 用于设置自增列缓存的刷新间隔。


|      **属性**      |    **描述**    |
|------------------|--------------|
| 参数类型             | 时间类型         |
| 默认值              | 3600s        |
| 取值范围             | \[100ms, +∞) |
| 是否重启 OBServer 生效 | 否            |
