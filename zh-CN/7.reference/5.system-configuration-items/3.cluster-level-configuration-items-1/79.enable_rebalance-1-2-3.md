enable_rebalance 
=====================================
#docslug#/oceanbase-database/oceanbase-database/V4.0.0/enable_rebalance-1-2-3
enable_rebalance 用于设置是否开启自动负载均衡的功能。


|      **属性**      |                                                    **描述**                                                     |
|------------------|---------------------------------------------------------------------------------------------------------------|
| 参数类型             | 布尔类型                                                                                                          |
| 默认值              | True                                                                                                          |
| 取值范围             | <li> True：开启   <li> False：不开启    |
| 是否重启 OBServer 生效 | 否                                                                                                             |
