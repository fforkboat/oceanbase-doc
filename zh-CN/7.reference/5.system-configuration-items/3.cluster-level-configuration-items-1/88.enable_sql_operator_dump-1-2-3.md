enable_sql_operator_dump 
=============================================
#docslug#/oceanbase-database/oceanbase-database/V4.0.0/enable_sql_operator_dump-1-2-3
enable_sql_operator_dump 用于设置是否允许 SQL 处理过程的中间结果写入磁盘以释放内存。


|      **属性**      |                                                 **描述**                                                 |
|------------------|--------------------------------------------------------------------------------------------------------|
| 参数类型             | 布尔类型                                                                                                   |
| 默认值              | True                                                                                                   |
| 取值范围             | <li> True   <li> False    |
| 是否重启 OBServer 生效 | 否                                                                                                      |
