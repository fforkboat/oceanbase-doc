enable_monotonic_weak_read 
===============================================
#docslug#/oceanbase-database/oceanbase-database/V4.0.0/enable_monotonic_weak_read-1-2-3
enable_monotonic_weak_read 用于设置是否开启单调读。


|      **属性**      |                                                 **描述**                                                 |
|------------------|--------------------------------------------------------------------------------------------------------|
| 参数类型             | 布尔类型                                                                                                   |
| 默认值              | False                                                                                                  |
| 取值范围             | <li> True   <li> False    |
| 是否重启 OBServer 生效 | 否                                                                                                      |
