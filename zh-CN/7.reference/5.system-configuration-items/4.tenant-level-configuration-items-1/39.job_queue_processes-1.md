job_queue_processes 
========================================
#docslug#/oceanbase-database/oceanbase-database/V4.0.0/job_queue_processes-1
`job_queue_processes` 用来设定每个租户上可以同时运行的任务数上限，目的是为了控制任务挤占租户的资源。


|        属性        |     描述     |
|------------------|------------|
| 参数类型             | 整型         |
| 默认值              | 1000       |
| 取值范围             | \[0,1000\] |
| 是否重启 OBServer 生效 | 否          |



设置 `job_queue_processes` 为 `0`，则表示定时任务不启动。

> **说明**
> 
> 该配置项当前仅在 Oracle 模式下生效。
