major_compact_trigger 
==========================================

major_compact_trigger 用于设置多少次转储触发一次全局合并。


|      **属性**      |    **描述**    |
|------------------|--------------|
| 参数类型             | 整型           |
| 默认值              | 0            |
| 取值范围             | \[0, 65535\] |
| 是否重启 observer 进程生效 | 否            |



当 major_compact_trigger 配置项的取值为 0 时，表示无论转储多少次也不会触发合并。
