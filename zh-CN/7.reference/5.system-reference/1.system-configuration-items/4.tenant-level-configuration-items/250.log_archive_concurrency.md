# log_archive_concurrency

`log_archive_concurrency` 用于设置日志归档的并发度。

<main id="notice" type='explain'>
  <h4>说明</h4>
  <p>该配置项从 V4.1.0 版本开始生效范围调整为租户级，取值范围调整为 [1,100]，取值大于 10 时默认以 10 生效。</p>
</main>

| **属性** | **描述** |
| --- | --- |
| 参数类型 | 整型 |
| 默认值 | 1 |
| 取值范围 | \[1,100] |
| 是否重启 OBServer 节点生效 | 否 |
