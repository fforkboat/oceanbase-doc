查询表里符合特定搜索条件的数据 
====================================



当要查询满足特定搜索条件的数据时，给 SELECT 查询语句增加一个 WHERE 子句即可。SQL 语句格式如下：

```sql
SELECT select_list FROM table_list 
WHERE query_condition
```



查询 2 号仓库第 5 区的订单，SQL 语句如下：

```javascript
SELECT * FROM ordr WHERE o_w_id=2 and o_d_id=5 ;
```
