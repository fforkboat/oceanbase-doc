# 创建数据库

本节主要介绍 MySQL 模式中数据库的创建语句及操作方法。

## 通过 SQL 语句创建数据库

您可以使用 `CREATE DATABASE` 语句创建数据库。

示例：

* 创建数据库 `test2`，并指定字符集为 `UTF8`。

  ```sql
  obclient> CREATE DATABASE test2 DEFAULT CHARACTER SET UTF8;
  Query OK, 1 row affected (0.00 sec)
  ```

* 创建读写属性的数据库 `test3`。

  ```sql
  obclient> CREATE DATABASE test3 READ WRITE;
  Query OK, 1 row affected (0.03 sec)
  ```

* 创建只读属性的数据库 `test4`。

  ```sql
  obclient> CREATE DATABASE test4 READ ONLY;
  Query OK, 1 row affected (0.03 sec)
  ```

  更多 `CREATE DATABASE` 语句相关的语法说明请参见 [CREATE DATABASE](../../../../400.development-reference/100.sql-syntax/200.common-tenant-of-mysql-mode/600.sql-statement-of-mysql-mode/1100.create-database-of-mysql-mode.md)。
