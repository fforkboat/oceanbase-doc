BIT_LENGTH 
===============================
#docslug#/oceanbase-database/oceanbase-database/V4.0.0/BIT_LENGTH


声明 
-----------------------

```sql
BIT_LENGTH(str)
```



说明 
-----------------------

返回字符串 `str` 的比特位长度。

示例 
-----------------------

```sql
obclient> SELECT BIT_LENGTH('oceanbase');
+-------------------------+
| BIT_LENGTH('oceanbase') |
+-------------------------+
|                      72 |
+-------------------------+
1 row in set 
```
