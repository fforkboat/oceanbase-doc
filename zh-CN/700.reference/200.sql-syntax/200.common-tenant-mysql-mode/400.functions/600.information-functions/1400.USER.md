USER 
=========================
#docslug#/oceanbase-database/oceanbase-database/V4.0.0/USER


声明 
-----------------------

```sql
USER()
```



说明 
-----------------------

返回当前的用户名与主机名，返回值使用的字符集为 utf8mb4。

示例 
-----------------------

```sql
obclient> SELECT USER();
+---------------------+
| USER()              |
+---------------------+
| root@xxx.xx.xxx.xxx |
+---------------------+
1 row in set 
```
