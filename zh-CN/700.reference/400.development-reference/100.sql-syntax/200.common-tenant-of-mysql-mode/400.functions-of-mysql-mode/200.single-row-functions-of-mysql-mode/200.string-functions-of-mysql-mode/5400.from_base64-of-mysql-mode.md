# FROM_BASE64

## 声明

```sql
FROM_BASE64(str)
```

## 说明

解码一个 base-64 编码的字符串，并以二进制字符串的形式返回。通常用来对  [TO_BASE64()](53.to_base64-of-mysql-mode.md) 编码的字符串进行解码。

## 示例

```sql
obclient [test]> SELECT FROM_BASE64('T0I=');
+---------------------+
| FROM_BASE64('T0I=') |
+---------------------+
| OB                  |
+---------------------+
1 row in set 
```
