# UNCOMPRESSED_LENGTH

## 声明

```sql
UNCOMPRESSED_LENGTH(str)
```

## 说明

返回压缩字符串在 [COMPRESS()](4.compress-of-mysql-mode.md) 函数压缩前的长度。

## 示例

```sql
obclient [test]> SELECT UNCOMPRESSED_LENGTH(COMPRESS('abc'));
+--------------------------------------+
| UNCOMPRESSED_LENGTH(COMPRESS('abc')) |
+--------------------------------------+
|                                    3 |
+--------------------------------------+
1 row in set
```
