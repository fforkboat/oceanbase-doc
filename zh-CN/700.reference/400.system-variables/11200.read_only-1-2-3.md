read_only 
==============================
#docslug#/oceanbase-database/oceanbase-database/V4.0.0/read_only-1-2-3
read_only 用于设置租户是否为只读模式。


| **属性** |                                                     **描述**                                                      |
|--------|-----------------------------------------------------------------------------------------------------------------|
| 参数类型   | bool                                                                                                            |
| 默认值    | 0                                                                                                               |
| 取值范围   | <li> 1：租户为只读模式   <li> 0：租户为读写模式    |
| 生效范围   | GLOBAL                                                                                                          |


> **说明**
> 
> 设置后，会影响该租户下的所有普通用户，不影响 root 权限用户。
