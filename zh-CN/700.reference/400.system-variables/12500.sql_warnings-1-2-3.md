sql_warnings 
=================================
#docslug#/oceanbase-database/oceanbase-database/V4.0.0/sql_warnings-1-2-3
sql_warnings 用于在出现告警时控制单行 INSERT 语句是否生成信息字符串。


| **属性** |                                                        **描述**                                                         |
|--------|-----------------------------------------------------------------------------------------------------------------------|
| 参数类型   | bool                                                                                                                  |
| 默认值    | OFF                                                                                                                   |
| 取值范围   | <li> ON：生成一个信息字符串   <li> OFF：不生成信息字符串    |
| 生效范围   | <li> GLOBAL   <li> SESSION               |
