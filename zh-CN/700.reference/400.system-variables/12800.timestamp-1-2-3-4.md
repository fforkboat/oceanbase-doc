timestamp 
==============================
#docslug#/oceanbase-database/oceanbase-database/V4.0.0/timestamp-1-2-3-4
TIMESTAMP 是时间戳，单位为秒，该变量影响 `select now(6) `的结果。


| **属性** |       **描述**       |
|--------|--------------------|
| 参数类型   | numeric            |
| 默认值    | 0                  |
| 取值范围   | \[0, 253402272000) |
| 生效范围   | SESSION            |
