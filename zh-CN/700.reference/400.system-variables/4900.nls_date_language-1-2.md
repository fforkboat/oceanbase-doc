nls_date_language 
======================================
#docslug#/oceanbase-database/oceanbase-database/V4.0.0/nls_date_language-1-2
`nls_date_language` 表示数据库国际化支持中的日期默认语言。用于 TO_DATE 和 TO_CHAR 输入或返回的日期、月份、上下午、公元前后等信息。

> **说明**
> 
> 该变量当前暂不生效。

| **属性**  |                                                   **描述**                                                   |
|---------|------------------------------------------------------------------------------------------------------------|
| 参数类型    | varchar                                                                                                    |
| 默认值     | AMERICAN                                                                                                   |
| 取值范围    | N/A                                                                                                        |
| 生效范围    | <li> Global   <li> Session    |
| 是否参与序列化 | 是                                                                                                          |
