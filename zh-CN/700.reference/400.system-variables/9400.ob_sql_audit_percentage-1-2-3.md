ob_sql_audit_percentage 
============================================
#docslug#/oceanbase-database/oceanbase-database/V4.0.0/ob_sql_audit_percentage-1-2-3
`ob_sql_audit_percentage` 用于设置 SQL Audit 内存上限占当前租户内存的百分比。


| **属性**  |  **描述**   |
|---------|-----------|
| 参数类型    | int       |
| 默认值     | 3         |
| 取值范围    | \[0, 80\] |
| 生效范围    | Global    |
| 是否参与序列化 | 是         |
