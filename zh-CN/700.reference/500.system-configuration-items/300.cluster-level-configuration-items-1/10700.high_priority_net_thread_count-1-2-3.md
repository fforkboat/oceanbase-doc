high_priority_net_thread_count
===================================================

#docslug#/oceanbase-database/oceanbase-database/V4.0.0/high_priority_net_thread_count-1-2-3
`high_priority_net_thread_count` 用于设置高优先级网络线程数。

|      **属性**      |   **描述**   |
|------------------|------------|
| 参数类型             | 整型         |
| 默认值              | 0          |
| 取值范围             | \[0, 64\] |
| 是否重启 OBServer 生效 | 是          |

该配置项的取值为 `0` 时，表示关闭该功能。
