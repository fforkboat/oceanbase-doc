memory_limit_percentage 
============================================
#docslug#/oceanbase-database/oceanbase-database/V4.0.0/memory_limit_percentage-1
memory_limit_percentage 用于设置系统总可用内存大小占总内存大小的百分比。


|      **属性**      |   **描述**   |
|------------------|------------|
| 参数类型             | 整型         |
| 默认值              | 80         |
| 取值范围             | \[10, 90\] |
| 是否重启 OBServer 生效 | 否          |
