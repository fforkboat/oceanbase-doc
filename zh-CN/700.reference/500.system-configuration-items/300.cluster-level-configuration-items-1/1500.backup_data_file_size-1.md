backup_data_file_size 
==========================================
#docslug#/oceanbase-database/oceanbase-database/V4.0.0/backup_data_file_size-1
`backup_data_file_size` 用于设置备份数据文件的容量。


|      **属性**      |   **描述**    |
|------------------|-------------|
| 参数类型             | 容量单位        |
| 默认值              | 4G          |
| 取值范围             | \[512M,4G\] |
| 是否重启 OBServer 生效 | 否           |
