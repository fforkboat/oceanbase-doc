mysql_port 
===============================
#docslug#/oceanbase-database/oceanbase-database/V4.0.0/mysql_port-1-2-3
mysql_port 用于设置 SQL 服务协议端口号。


|      **属性**      |     **描述**      |
|------------------|-----------------|
| 参数类型             | 整型              |
| 默认值              | 2881            |
| 取值范围             | \[1025, 65535\] |
| 是否重启 OBServer 生效 | 否               |
