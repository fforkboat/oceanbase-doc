server_check_interval 
==========================================
#docslug#/oceanbase-database/oceanbase-database/V4.0.0/server_check_interval-1-2-3
server_check_interval 用于设置 Server 表一致性检查的时间间隔。


|      **属性**      |  **描述**   |
|------------------|-----------|
| 参数类型             | 时间类型      |
| 默认值              | 30s       |
| 取值范围             | \[1s, +∞) |
| 是否重启 OBServer 生效 | 否         |
