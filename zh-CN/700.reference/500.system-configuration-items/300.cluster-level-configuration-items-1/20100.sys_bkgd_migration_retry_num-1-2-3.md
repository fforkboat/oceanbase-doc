sys_bkgd_migration_retry_num 
=================================================
#docslug#/oceanbase-database/oceanbase-database/V4.0.0/sys_bkgd_migration_retry_num-1-2-3
sys_bkgd_migration_retry_num 用于设置副本迁移失败时最多重试次数。


|      **属性**      |   **描述**   |
|------------------|------------|
| 参数类型             | 整型         |
| 默认值              | 3          |
| 取值范围             | \[3, 100\] |
| 是否重启 OBServer 生效 | 否          |
