|      **属性**      |                                                                                                                                                **描述**                                                                                                                                                |
|------------------|------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------|
| 参数类型             | 字符串类型                                                                                                                                                                                                                                                                                                |
| 默认值              | none                                                                                                                                                                                                                                                                                                 |
| 取值范围             | <li> none   <li> lz4_1.0   <li> snappy_1.0   <li> zlib_1.0   <li> zstd_1.0   <li> zstd 1.3.8    |
