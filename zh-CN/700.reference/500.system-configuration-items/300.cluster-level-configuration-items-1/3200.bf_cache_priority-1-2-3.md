bf_cache_priority 
======================================
#docslug#/oceanbase-database/oceanbase-database/V4.0.0/bf_cache_priority-1-2-3
bf_cache_priority 用于设置 Bloom Filter 缓存优先级。


|      **属性**      |  **描述**  |
|------------------|----------|
| 参数类型             | 整型       |
| 默认值              | 1        |
| 取值范围             | \[1, +∞) |
| 是否重启 OBServer 生效 | 否        |
