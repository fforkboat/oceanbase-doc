# nls_date_format

`nls_date_format` 用于控制 date 类型转 str 的格式，以及 str 隐式转 date 的格式。

  <main id="notice" type='explain'>
    <h4>说明</h4>
    <p>该变量当前暂不生效。</p>
  </main>

| **属性**  |                                                   **描述**                                                   |
|---------|------------------------------------------------------------------------------------------------------------|
| 参数类型    | varchar                                                                                                    |
| 默认值     | DD-MON-RR                                                                                                  |
| 取值范围    | N/A                                                                                                        |
| 生效范围    | <li> Global   <li> Session    |
| 是否参与序列化 | 是                                                                                                          |
