# ob_enable_show_trace

ob_enable_show_trace 用于设置是否使用 show trace 日志。

<main id="notice" type='explain'>
 <h4>说明</h4>
 <p><code>show trace</code> 功能在 4.0 版本暂不支持，4.1 版本支持。</p>
</main>

| **属性**  |                                                   **描述**                                                   |
|---------|------------------------------------------------------------------------------------------------------------|
| 参数类型    | bool                                                                                                       |
| 默认值     | 0                                                                                                          |
| 取值范围    | <li> 0：不开启</li>   <li> 1：开启</li>        |
| 生效范围    | <li> GLOBAL</li>   <li> SESSION</li>    |
| 是否参与序列化 | 是                                                                                                          |
