# tx_isolation

tx_isolation 用于设置事务隔离级别。

| **属性**  |                                                                                                               **描述**                                                                                                               |
|---------|------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------|
| 参数类型    | varchar                                                                                                                                                                                                                            |
| 默认值     | READ-COMMITTED                                                                                                                                                                                                                     |
| 取值范围    | <li> READ-UNCOMMITTED   <li> READ-COMMITTED   <li> REPEATABLE-READ   <li> SERIALIZABLE    |
| 生效范围    | <li> GLOBAL   <li> SESSION                                                                                                                            |
| 是否参与序列化 | 是                                                                                                                                                                                                                                  |
