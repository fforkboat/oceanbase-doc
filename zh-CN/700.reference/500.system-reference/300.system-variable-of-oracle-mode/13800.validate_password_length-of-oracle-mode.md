# validate_password_length

`validate_password_length` 设置用户密码最小长度。

  <main id="notice" type='explain'>
    <h4>说明</h4>
    <p>该变量当前暂不生效。</p>
  </main>

| **属性**  |      **描述**      |
|---------|------------------|
| 参数类型    | uint             |
| 默认值     | 0                |
| 取值范围    | \[0,2147483647\] |
| 生效范围    | Global           |
| 是否参与序列化 | 是                |
