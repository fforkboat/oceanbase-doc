# validate_password_mixed_case_count

`validate_password_mixed_case_count` 用于设置用户密码至少包含的大写字母个数以及至少包含的小写字母个数。

  <main id="notice" type='explain'>
    <h4>说明</h4>
    <p>该变量当前暂不生效。</p>
  </main>

| **属性**  |      **描述**      |
|---------|------------------|
| 参数类型    | uint             |
| 默认值     | 0                |
| 取值范围    | \[0,2147483647\] |
| 生效范围    | Global           |
| 是否参与序列化 | 是                |
