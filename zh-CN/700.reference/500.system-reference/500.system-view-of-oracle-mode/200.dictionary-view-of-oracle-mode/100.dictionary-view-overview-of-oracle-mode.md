# 字典视图总览

本文根据字典视图的功能，详细地罗列了 OceanBase 数据库的字典视图。

  <main id="notice" >
    <h4>功能适用性</h4>
    <p>该内容仅适用于 OceanBase 数据库企业版。OceanBase 数据库社区版仅提供 MySQL 模式。</p>
  </main>
