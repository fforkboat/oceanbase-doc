# oceanbase.DBA_SCHEDULER_JOBS

## 功能

展示数据库中所有调度程序作业的信息。

<main id="notice" type='explain'>
  <h4>说明</h4>
  <p>从 V4.0.0 版本开始引入。</p>
</main>
